# イオン-電子緩和（P_ie）とαチャネリング

> **一言で:** p-B11の真のボトルネックは制動放射ではなく、イオンから電子へのエネルギー流出（P_ie）。αチャネリングはそれを迂回する唯一の候補。

---

## 1. P_ie とは何か

### 熱いイオンと冷たい電子

p-B11核融合では、イオン温度 Ti ≈ 300 keV（約30億℃）が必要。
しかしプラズマ中の電子はイオンと衝突（クーロン衝突）してエネルギーを受け取る。

$$
P_{ie} = \frac{n_e (T_i - T_e)}{\tau_{ie}}
$$

これは「熱い物体と冷たい物体を接触させると熱が移動する」のと同じ。止められない。

### なぜ p-B11 で致命的か

緩和率は $Z^2$ に比例する。ホウ素は Z=5：

| 燃料 | Z_eff | Z²効果 | P_ie の相対値 |
|------|-------|--------|--------------|
| D-T | 1.0 | 1 | 1×（基準） |
| D-³He | 1.6 | 2.6 | ~3× |
| **p-B11** | **4.33** | **18.8** | **~25×** |

p-B11では、イオンのエネルギーが電子に流出する速度がD-Tの **25倍** 速い。

### 数字で見るとどれくらいひどいか

arXiv:2405.13260（2024年）の計算によると、Ti = 300 keV の条件で：

```
P_fusion（核融合出力）       = 1.0 （基準化）
P_ie（イオン→電子の流出）    = 10〜20  ← !!
P_brem（制動放射損失）       = ~1.0
```

**核融合で1のエネルギーを得る間に、イオンから電子へ10〜20のエネルギーが流出する。**

電子が温まると制動放射（P_brem）が増える → さらにエネルギーが失われる → 悪循環。

### 水槽のたとえ

```
  【イオン水槽】Ti = 300 keV（お湯）
       │
       │ ← P_ie: 太いパイプ（Z²=18.8で25倍太い）
       │         止めることができない
       ▼
  【電子水槽】Te = ?? keV
       │
       │ ← P_brem: 制動放射として外に漏れる
       ▼
     【宇宙】（エネルギー損失）

  核融合反応 → α粒子（3個、合計8.7 MeV）
       │
       ├→ 電子に衝突してエネルギーを渡す（通常）
       │   → 電子がさらに温まる → P_brem 増加
       │
       └→ イオンにエネルギーを渡す（αチャネリング）
           → イオン水槽を補充 → P_ie に対抗できる
```

## 2. なぜ P_fus/P_brem ≈ 1.0 でも足りないのか

### ベースラインの更新

Rider（1995）の P_fus/P_brem ≈ 0.22 は旧データだった。最新の研究で：

| 研究 | 年 | P_fus/P_brem |
|------|---|---|
| Rider | 1995 | 0.22 |
| Nevins & Swain | 2000 | 0.9〜1.05 |
| Putvinski et al. | 2019 | ~1.03 |

P_fus/P_brem ≈ 1.0 なら制動放射は相殺できるはず。**では何が問題か？**

### P_ie が問題

P_fus/P_brem = 1.0 は「核融合出力 = 制動放射損失」を意味する。
しかし **損失は制動放射だけではない：**

```
核融合出力 P_fus    =  1.0

損失:
  P_brem（制動放射）  =  1.0   ← P_fus/P_brem = 1.0 で相殺
  P_ie（イオン→電子） = 10〜20 ← これが全然足りない！
  P_sync（シンクロトロン）= 0.1〜1.0
  P_transport（輸送損失）= ???

収支:  1.0 - 1.0 - 15 - 0.5 = -15.5  ← 大赤字
```

P_ie を外部加熱で補償するには、核融合出力の10〜20倍の電力を投入し続ける必要がある。
これでは発電所ではなく電力消費施設になる。

## 3. αチャネリングとは何か

### 発想

p-B11核融合は3つのα粒子（ヘリウム原子核）を生成する。各2.9 MeV、合計8.7 MeV。

通常、α粒子はプラズマ中で **電子にエネルギーを渡す**（α粒子は重いが、電子との衝突が多い）。
これは最悪の結果：電子が温まり、P_brem が増える。

**αチャネリングの発想：α粒子のエネルギーを電子ではなくイオンに直接渡す。**

### 仕組み（Fisch & Rax, 1992; Ochs & Fisch, 2024）

```
通常のα粒子減速:
  α粒子 → 電子に衝突 → 電子が加熱 → 制動放射増加
  （イオンにはほとんどエネルギーが渡らない）

αチャネリング:
  α粒子 → 波動（プラズマ波）を励起 → 波動がイオンを加速
  （電子を飛ばしてイオンに直接エネルギーを渡す）
```

これは物理的には：
1. α粒子がプラズマ中の特定の波動モード（イオンバーンシュタイン波など）と共鳴する
2. α粒子の運動エネルギーが波動のエネルギーに変換される
3. 波動がイオンに吸収され、イオンが加速される

**電子を「バイパス」してイオンに直接エネルギーを届ける。**

### なぜこれがP_ie問題の解決になるか

```
αチャネリングなし:
  核融合 → α粒子 → 電子 → 制動放射で損失
  P_ieでイオンのエネルギーも電子に吸い取られる
  → 大赤字

αチャネリングあり:
  核融合 → α粒子 → 波動 → イオン
  イオンのエネルギーが補充される
  P_ieで失った分をα粒子が補填する
  → 赤字幅が大幅縮小
```

### 定量評価（楽観的シナリオ）

αチャネリング効率80%（α粒子エネルギーの80%がイオンに転送）の場合：

| パワー項目 | αチャネリングなし | αチャネリングあり |
|-----------|-----------------|-----------------|
| P_fus | 1.0 | 1.0 |
| α → 電子 | 1.0（全額） | 0.2（20%のみ） |
| α → イオン | 0.0 | 0.8（80%） |
| P_ie（イオン→電子） | ~15 | ~2（Te低下により大幅減） |
| P_brem | ~1.0 | ~0.25（Te低下） |
| **収支** | **-15.5** | **-1.3** |

**赤字が-15.5 → -1.3に改善。** ただし **まだ赤字。**

直接エネルギー変換（α粒子を直接電気に変換、効率70%）と制動放射X線回収（効率20%）を組み合わせても、完全な黒字化は困難。

## 4. αチャネリングの現状

### 理論的基盤

| 論文 | 内容 |
|------|------|
| Fisch & Rax (1992) | αチャネリングの概念を提唱 |
| Fisch (1999, NF) | トカマクでの応用理論 |
| Ochs & Fisch (2024, PoP 31) | FRCでの高速陽子へのエネルギー転送理論。ARPA-E資金 |

### 実験的検証

**存在しない。**

αチャネリングは2024年時点で **紙の上にしか存在しない** 理論メカニズムである。

- 波動-α粒子共鳴の実験的観測: なし
- 波動→イオンエネルギー転送の定量測定: なし
- FRCプラズマでのαチャネリング実験: 計画段階

### TAE Technologiesの位置づけ

TAEはFRC方式でp-B11を目指す最も進んだプロジェクト。
C-2W装置でTi/Te ≈ 4（keV領域）を達成している。
しかし：
- 現在の温度: ~7000万℃
- p-B11に必要な温度: ~30億℃（**43倍のギャップ**）
- αチャネリングの実験はまだ行っていない

## 5. 正直な評価

### p-B11の成否はαチャネリングに懸かっている

```
αチャネリングが実証される → p-B11に「狭い窓」が開く（限定的赤字の延命戦略）
αチャネリングが不可能     → p-B11は発電燃料として不可能（ニッチ用途は残る）
```

### D-Tとの本質的な違い

| | p-B11 | D-T |
|---|---|---|
| 最大の壁 | P_ie（物理定数Z=5に根ざす） | 材料・トリチウム（工学問題） |
| 解決手段 | αチャネリング（**未実証の物理**） | 照射試験・ブランケット実証（**既知の方法**） |
| 不確実性の種類 | 「解けるかどうか分からない」 | 「時間がかかるが解ける」 |

### ホウ素の原子番号は変えられない

> $Z_{eff}^2 = 18.8$ は物理定数である。
> ホウ素の陽子を1個減らしてベリリウム（Z=4）にすることはできない。
> p-B11を選ぶ限り、この18.8倍の不利は永久に付きまとう。
> αチャネリングはこの不利を「迂回」する試みであり、「解消」するものではない。
