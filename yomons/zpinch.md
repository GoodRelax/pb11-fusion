# Zピンチ — 電流が自分自身を圧縮する原理

> **対応アニメーション:** [zpinch.html](https://goodrelax.github.io/pb11-fusion/simulations/zpinch.html)
> **関連:** [compression-topology.md](compression-topology.md) / [maglif.md](maglif.md) / [overview.md](overview.md)

---

## 一言でいうと

> プラズマに大電流を流すと、その電流自身が作る磁場がプラズマを内側に押しつぶす。
> これが **Zピンチ（Z-pinch）** 。外部磁石なし、電流だけで圧縮できる最もシンプルな核融合方式。

---

## 原理：なぜ圧縮されるのか

### ステップ1：電流を流す

プラズマ（電離ガス）の柱にZ方向（軸方向）に大電流 $I$ を流す。

```
    陽極 (+)
    ──────────
    ↑ ↑ ↑ ↑ ↑    電流 I （Z方向、上向き）
    │ │ │ │ │
    ↑ ↑ ↑ ↑ ↑
    ──────────
    陰極 (−)
```

### ステップ2：磁場が生まれる

電流が流れると、右ねじの法則でプラズマの周囲にリング状の磁場 $B_\theta$ が生じる。

$$
B_\theta = \frac{\mu_0 I}{2\pi r}
$$

- $r$：中心からの距離
- 中心に近いほど磁場が強い

```
        B_θ（リング状の磁場）
        ╭──→──╮
        │      │
   I ↑  │  ●  │  ← プラズマ柱の断面
        │      │
        ╰──←──╯
```

### ステップ3：ローレンツ力が内向きに押す

電流 $\vec{J}$ と磁場 $\vec{B}_\theta$ のクロス積で、**内向きのローレンツ力**が生まれる。

$$
\vec{F} = \vec{J} \times \vec{B}
$$

- $\vec{J}$：Z方向（上向き）
- $\vec{B}$：θ方向（リング状）
- $\vec{F}$：**−r 方向（内向き）** ← これが圧縮力

```
    F ←──── ● ────→ F
            │
    F ←──── ● ────→ F      全方向から中心に向かって押される
            │
    F ←──── ● ────→ F
```

### ステップ4：圧縮 → 加熱

プラズマが圧縮されると、断熱圧縮の法則で温度が上がる。

$$
T \propto \rho^{\gamma - 1}
$$

圧縮比が十分大きければ、核融合温度（数十億℃）に到達する。

---

## 磁気圧力 vs プラズマ圧力

Zピンチは「磁気圧力」と「プラズマ圧力」のバランスで理解できる。

$$
\frac{B_\theta^2}{2\mu_0} = n k_B T
$$

| 項 | 意味 |
|---|---|
| 左辺（磁気圧力） | 電流が作る磁場の圧力。**外から押す力。** |
| 右辺（プラズマ圧力） | プラズマの熱運動。**内から押す力。** |

平衡条件（Bennett条件）：

$$
I^2 = \frac{8\pi N k_B (T_i + T_e)}{\mu_0}
$$

- $N$：単位長さ当たりの粒子数
- 電流 $I$ を増やす → 磁気圧力が増える → より強く圧縮 → より高温

## 車のエンジンで例えると

| Zピンチ | ガソリンエンジン |
|---|---|
| プラズマ柱 | シリンダー内の混合気 |
| 電流による磁場 | ピストンの圧縮 |
| ローレンツ力 | ピストンが押す力 |
| 断熱圧縮で加熱 | 圧縮行程で温度上昇 |
| 核融合点火 | スパークプラグの点火 |
| α粒子の飛散 | 爆発（膨張行程） |

違い：Zピンチは**全方向から同時に**圧縮する（ピストンは一方向だけ）。

## 致命的な弱点：不安定性

Zピンチは原理がシンプルだが、**2つの致命的な不安定性**がある。

### ソーセージ不安定性（m=0）

プラズマ柱の一部がくびれると、くびれた部分で磁場が強くなり、さらにくびれが加速する。

```
    ====●●●●●●====    正常
    ==●●●●●●●●●●==    ← 少しくびれる
    ●●●●●●●●●●●●●●    ← くびれが加速
    ==  ●●●●  ==      ← 分裂！
```

ソーセージのくびれた形に似ているのでこう呼ぶ。

### キンク不安定性（m=1）

プラズマ柱が横に曲がると、曲がった外側で磁場が弱くなり、内側で強くなるため、さらに曲がりが加速する。

```
    │         │
    │    →    ╲
    │         ╱    ← 曲がり始めると止まらない
    │    →    ╲
    │         │
```

### 時間スケール

両方とも $\sim 10^{-7}$ s（100 ns）程度で成長する。Zピンチの圧縮時間と同程度なので、**圧縮が完了する前に崩壊する**。

---

## 不安定性への対策

| 対策 | 方式 | 代表例 |
|---|---|---|
| 軸方向磁場 $B_z$ を追加 | 磁力線の張力で曲がりを抑制 | **MagLIF**（Sandia） |
| 剪断流 | プラズマの流れで不安定性の成長を抑制 | **Zap Energy** |
| 超短パルス | 不安定性が成長する前に燃やし切る | **本プロジェクト（4サイクル）** |
| ライナー（金属筒） | 固体の慣性で崩壊を遅らせる | **MagLIF** |

4サイクル方式の戦略は3番目：**不安定性の成長時間（~100 ns）より短い燃焼時間（~1-10 ns）で逃げ切る。**

---

## 4サイクル方式での役割

4サイクル方式では、Zピンチは **Phase 2（圧縮行程）** で使われる。

```
Phase 1: 電子ビームでプラズマ生成・予熱（体 → 線）
Phase 2: Zピンチで断熱圧縮（線 → 点）  ← ★ここ
Phase 3: 核融合反応（燃焼）
Phase 4: α粒子回収・発電（膨張）
```

### なぜZピンチを選ぶか

| 利点 | 内容 |
|---|---|
| 外部磁石不要 | 電流だけで圧縮 → 構造がシンプル |
| 超高電流が可能 | MA級のパルス電流は既存技術（Sandia Zマシン：26 MA） |
| 超短パルスと相性が良い | ナノ秒パルスでTe上昇を抑制 → 制動放射対策 |
| 「体→線→点」の自然な経路 | 円柱プラズマが軸上に集中 → 圧縮トポロジーRoute C |

### 正直な限界

| 限界 | 内容 |
|---|---|
| 不安定性 | MHD不安定性が~100 nsで成長。超短パルスで逃げ切れるか未実証 |
| 繰り返し | 毎パルスで電極がダメージを受ける。10⁶ショット以上の耐久性が未達 |
| 電子加熱 | Zピンチ電流は電子も加熱する（オーム加熱）→ Te上昇 → 制動放射増 |
| 電子排出との矛盾 | 電子がZピンチ電流を運んでいる → 電子を抜くと電流が弱まる |

## 歴史

| 年代 | 出来事 |
|---|---|
| 1934年 | Bennett が平衡条件を導出 |
| 1950年代 | 米英ソが核融合用Zピンチを実験。不安定性で断念 |
| 1970〜80年代 | X線源・核実験シミュレーション用途に転用 |
| 2000年代〜 | Sandia Zマシン（26 MA）で核融合研究再開 |
| 2012年 | MagLIF（Zピンチ + 磁場 + レーザー予熱）で中性子観測 |
| 2020年代〜 | Zap Energy が剪断流Zピンチで商用化を目指す |

Zピンチは「最初に試され、最初に捨てられ、最近また拾われた」方式。
不安定性を制御する技術が進歩したことで、再評価されている。
