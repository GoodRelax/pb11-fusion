## 1. TAE Technologies

**方式:** FRC（磁場反転配位）+ NBI（中性ビーム入射）

### 電子対策

```
主戦略: FRC 高β閉じ込めで Ti >> Te の非平衡プラズマを作る
         + 直接エネルギー変換でα粒子の運動エネルギーを回収
         + 将来的にαチャネリングを導入（Ochs & Fisch 2024）
```

| 対策 | 内容 | 状態 |
|------|------|------|
| Ti/Te 非平衡 | NBIでイオンを選択加熱。C-2WでTi/Te ≈ 4達成 | 🟡 keV領域で達成。300 keVには43倍のギャップ |
| FRC 高β | β~50-90%で外部磁場を低く抑え、シンクロトロン放射を低減 | 🟡 FRC安定化は実証済み。p-B11条件は未達 |
| 直接変換 | 磁気ノズル端からα粒子を直接電気に変換（η~70%） | 🟡 概念実証段階 |
| αチャネリング | α→波動→イオンのエネルギー転送。P_ie問題の根本解決 | 🔴 理論のみ |

### 実績

- **p-B11核融合の初計測成功**（2023, Nature Communications, TAE + NIFS共同）
- C-2Wで安定FRC閉じ込め30ms、Ti+Te > 3 keV
- 2025: NBIのみでFRC生成・加熱・安定化に成功

### 正直な評価

| 項目 | 値 |
|------|-----|
| 達成温度 | ~7000万℃ |
| 必要温度 | ~30億℃ |
| **ギャップ** | **43倍** |
| P_ie問題への解 | αチャネリング（未実証） |
| Q > 1への距離 | 数段階 |
| 検証状況 | 🟢 査読論文多数 |
| 資金 | 60億ドル超（Trump Media合併） |
| 見込み | **最も進んでいるが、p-B11到達は2030年代以降。P_ie問題が未解決。** |

## 2. HB11 Energy

**方式:** ペタワット級レーザーで陽子を加速 → ホウ素ターゲットに衝突

### 電子対策

```
主戦略: 制動放射問題を「構造的に回避」する
         → 熱平衡プラズマを作らない
         → レーザーで非熱的にイオンを加速して反応させる
         → 電子は加熱されるが、反応は電子温度に依存しない
```

| 対策 | 内容 | 状態 |
|------|------|------|
| 非熱核融合 | レーザーでMeV陽子を生成、ホウ素と非熱的に反応 | 🟡 α粒子生成を確認 |
| アバランシェ反応 | α粒子が二次反応を起こし連鎖的に増倍 | 🔴 理論予測10³-10⁴倍、実験で10-100倍 |

### 正直な評価

| 項目 | 値 |
|------|-----|
| α粒子生成 | ~10¹¹ /ショット |
| ブレークイーブンに必要 | ~10¹⁵ /ショット |
| **ギャップ** | **10⁴倍** |
| P_ie問題への解 | 回避（熱プラズマを使わない） |
| 新たなボトルネック | レーザー効率（壁コンセント効率10-30%） |
| 検証状況 | 🟡 査読論文あり、独立追試は限定的 |
| 見込み | **制動放射は回避できるが、レーザー効率がQ>1を阻む。10⁴倍のギャップは巨大。** |

**HB11の本質的な問い:** 制動放射は回避できても、レーザーの非効率性が新たな「電子問題」を生む。レーザーエネルギーの大部分は電子加熱に消費されるため、結局電子にエネルギーを奪われる構造は変わらない。

## 3. LPPFusion

**方式:** Dense Plasma Focus（DPF）のピンチで超高温・超高密度プラズマを生成

### 電子対策

```
主戦略: ギガテスラ級磁場で電子をランダウ量子化
         → 量子制動放射抑制で P_brem を 1/10-1/100 に
```

| 対策 | 内容 | 状態 |
|------|------|------|
| 量子磁場効果 | GT級磁場でランダウ準位が量子化 → 位相空間制限 → 制動放射抑制 | 🔴 核心主張が未検証 |
| 超高温イオン | DPFで平均イオンエネルギー > 200 keV を達成 | 🟡 イオン温度は達成 |

### 致命的な問題

**必要磁場 vs 実測磁場:**

```
量子制動放射抑制に必要: ~4 GT（ギガテスラ = 10⁹ T）
DPFピンチの実測磁場:    ~10²-10³ T
ギャップ:                6-7桁（100万〜1000万倍）
```

### 正直な評価

| 項目 | 値 |
|------|-----|
| イオン温度 | > 200 keV（p-B11に必要な領域） |
| GT級磁場の実証 | ❌ **実測で6-7桁不足** |
| P_ie問題への解 | 量子抑制で制動放射を消す（理論） |
| 検証状況 | 🟡 イオン温度の論文あり。量子磁場効果は独立検証なし |
| 見込み | **核心主張（GT級磁場）が物理的に非現実的。独立検証なし。楽観的すぎる。** |

## 5. LINEA Innovations

**方式:** FRC + タンデムミラーのハイブリッド + ビーム駆動非熱核融合

### 電子対策

```
主戦略: 「非熱核融合」— 熱平衡プラズマを作らない
         → 高エネルギー水素ビーム（数百keV）をホウ素プラズマに入射
         → ビームイオンが直接核反応を起こす
         → プラズマ全体を数十億度に加熱する必要がない
         + ミラー端損失で電子が選択的に逃げる（軽いため）
         + FRC高βでシンクロトロン放射を低減
         + 直接エネルギー変換（α粒子を磁気ノズルで回収）
```

| 対策 | 内容 | 状態 |
|------|------|------|
| 非熱核融合（ビーム駆動） | 数百keVビームで直接反応。バルク加熱不要 | 🟡 原理は既知。p-B11でのQ>1は未実証 |
| ミラー端損失 | 開放端から軽い電子が先に逃げる → 自然なTi/Te分離 | 🟡 アンビポーラ電位が制限。定量評価未公開 |
| FRC高β | β>50%で外部磁場低減 → シンクロトロン放射抑制 | 🟡 TAEと同じ利点 |
| 直接変換 | α粒子を開放磁力線で誘導し直接電力変換 | 🟡 概念段階 |

### TAEとの違い

| | TAE | LINEA |
|---|---|---|
| 基盤技術 | FRC + NBI | FRC + タンデムミラー + ビーム |
| 加熱方式 | NBIでプラズマ全体を段階加熱 | ビーム粒子が直接反応（非熱） |
| 電子対策の核心 | αチャネリング（将来） | ミラー端損失 + 非熱 |
| プラズマ温度 | Ti+Te > 3 keV達成 | 未公開 |
| ルーツ | UCI（カリフォルニア大） | 日大 + 筑波大（GAMMA 10） |

### P_ie問題への回答

LINEAの「非熱核融合」アプローチは、**バルクプラズマの温度に依存しない** 点がユニーク。ビームイオン（数百keV）がターゲットプラズマ中のホウ素と直接反応するため、プラズマ全体をTi=300keVに加熱する必要がない。

しかし：
- ビームイオンはプラズマ中で**電子にエネルギーを失う**（電子阻止能、beam-target.mdと同じ問題）
- ビームの減速時間 vs 核反応時間の競争は、バルクのτ_eq vs τ_burnの競争と本質的に同じ
- **P_ie問題を回避しているのではなく、形を変えているだけ** の可能性がある

### 正直な評価

| 項目 | 値 |
|------|-----|
| 方式の独自性 | FRC+タンデムミラーのハイブリッドは独自。日本のGAMMA 10の知見を活用 |
| 実験結果 | **未公開**（FRC衝突合体の実証は日大で達成） |
| P_ie問題への回答 | ミラー端損失 + 非熱。定量評価は未公開 |
| 技術ページの情報量 | 原理説明は丁寧だが、定量データ（温度・密度・閉じ込め時間）なし |
| 検証状況 | 🔴 査読論文なし（技術ページのみ） |
| 見込み | **原理は興味深いが、定量的裏付けが全くない段階。TAEとの差別化ポイント（ミラー端損失による電子排出）の定量評価が鍵。** |

### 全員に共通する問題

**P_ie（イオン-電子緩和）を正面から解決しているプロジェクトは存在しない。**

- TAE: αチャネリングに将来的に期待（未実証）
- HB11: 熱プラズマを避けることでP_ieを回避（しかしレーザー効率が新ボトルネック）
- LPP: 量子効果で制動放射を消す主張（物理的に疑問）
- Marvel/4サイクル: P_ie問題にまだ到達していない（それ以前の段階）

**αチャネリングの実験的検証が、p-B11核融合全体の生死を分ける。**
これはTAEだけの問題ではなく、p-B11を目指す全プロジェクトに共通する根本課題。
