## Helion のアプローチ: FRC 衝突合体 + 磁気圧縮

### ステップ1: FRC プラズモイドの形成

装置の両端で、それぞれ独立した FRC（磁場反転配位）プラズモイドを形成する。
TAE Technologies と同じく FRC を使うが、Helion は **2つ作る** のがポイント。

```
左端:                    右端:
 ←←←←←                  ←←←←←
 →→→→→  FRC-L            →→→→→  FRC-R
 ←←←←←                  ←←←←←
  ↑                        ↑
  プラズモイド               プラズモイド
```

### ステップ2: FRC 衝突合体

2つの FRC を磁気加速で中央に向かって **超音速で衝突させる**。

```
  FRC-L  →→→  |  ←←←  FRC-R
               ↓
         衝突・合体！
               ↓
          合体 FRC
    （より高温・高密度）
```

衝突のエネルギーで一気にイオンを加熱。NBI のような外部加熱装置は不要。
これは **運動エネルギー → 熱エネルギー** の直接変換。

### ステップ3: 磁気圧縮

合体した FRC を**外部コイルの磁場で圧縮**する。

```
圧縮前:  [  ======  ]  ← 合体 FRC（大きい）
             ↓ 外部コイル電流 UP
圧縮後:  [  ===  ]     ← 圧縮 FRC（小さい・高温・高密度）
```

圧縮で温度と密度が上がり、D-³He 核融合反応が起きる:

```
D + ³He → ⁴He (3.6 MeV) + p (14.7 MeV)
```

生成物は荷電粒子のみ（中性子なし！…理想的には）。

### ステップ4: 直接発電（誘導型）

ここが Helion の最大の独自性。核融合で膨張するプラズマが**磁束を変化させ**、
外部コイルに電流を誘導する。

```
原理: ファラデーの法則
  EMF = -dΦ/dt

  ① 圧縮時: コイル → プラズマ にエネルギーを投入
  ② 膨張時: プラズマ → コイル にエネルギーを回収
  ③ 核融合エネルギー分だけ、回収 > 投入 → 正味発電
```

**これは静電型直接変換ではなく、誘導型直接変換。**

- 静電型: 荷電粒子を減速してエネルギー回収（TAE が検討）
- 誘導型: 磁束変化で電流誘導（Helion のアプローチ）← **はるかにシンプル**

効率目標は 95% 以上。原理的にカルノー効率に縛られない。

## パルス運転

Helion は **~1 Hz**（1秒に1回）のパルス運転を前提とする。

```
サイクル:
  0 ms     : FRC 形成（両端）
  ~0.5 ms  : 加速・衝突
  ~1 ms    : 磁気圧縮 + 核融合
  ~2 ms    : 膨張 + エネルギー回収
  ~5 ms    : 排気・リセット
  ...
  ~1000 ms : 次のサイクル開始
```

パルス運転の利点:
- プラズマを維持し続ける必要がない → 安定性の問題が軽減
- 各サイクルでリセットできる → 不純物蓄積の問題なし
- 繰り返し率を上げることで平均出力を確保

## D-³He 燃料の正直な評価

### メリット: 制動放射が軽い

```
p-B11:  Z_eff² ≈ (1×5 + 5×1)²/(1+5)² ≈ ... 実効的に重い
D-³He:  Z_eff² ≈ 2.8  ← p-B11 よりはるかに軽い！
```

D-³He は水素同位体（Z=1）とヘリウム（Z=2）の組み合わせなので、
**制動放射の問題が p-B11 よりはるかに小さい**。
p-B11 の最大ボトルネック（L1-2）が D-³He では大幅に緩和される。

### デメリット1: ³He の入手性（showstopper）

**³He は地球上にほとんど存在しない。**

```
³He の供給源:
  ① トリチウム崩壊（原子炉副産物）: ~15 kg/年（世界全体）
  ② 月面レゴリス: 大量にあるが採掘は SF
  ③ 天然ガス: 極微量

1 GW 発電所に必要な ³He: ~100 kg/年（推定）
現在の世界生産量: ~15 kg/年

→ 1基の発電所すら動かせない
```

これが D-³He 核融合の **根本的な showstopper**。
燃料がなければ、どんなに美しい物理でも発電所にはならない。

### デメリット2: 副反応としての D-D 反応

D-³He 燃料でも D が存在する限り、D-D 副反応が起きる:

```
D + D → T + p      (50%)
D + D → ³He + n    (50%)  ← 中性子が出る！
```

**「中性子フリー」は厳密には嘘。** D-D 副反応で一部中性子が発生する。
さらに生成された T は D-T 反応を起こし、14.1 MeV 中性子も出る。

完全に中性子フリーなのは p-B11 だけ。

## Helion vs 4サイクル案: 類比と対比

| 項目 | Helion | 4サイクル案 |
|------|--------|-----------|
| 燃料 | **D-³He** | **p-B11** |
| 閉じ込め | FRC（磁気） | Zピンチ（ハイブリッド） |
| 運転モード | **パルス（~1 Hz）** | **パルス（ns スケール）** |
| 圧縮 | 磁気圧縮（外部コイル） | 自己圧縮（Zピンチ） |
| 発電 | **直接発電（誘導型）** | **直接発電** |
| Ti/Te 問題 | D-³He なので深刻でない | 最大の課題（Rider の壁） |
| 制動放射 | Z_eff² ≈ 2.8 → 軽い | Z_eff 大 → 深刻 |
| 中性子 | D-D 副反応で一部あり | ほぼゼロ（理想的） |
| 燃料入手 | **³He がない（致命的）** | p と B → 無尽蔵 |
| 実験実績 | Trenta で 100M°C | なし（思考実験） |
| 資金 | ~900億円+ | 0円 |

### 共通点（意外に多い）
- **パルス運転** → 定常維持の難しさを回避
- **直接発電** → 熱サイクルを経由しない
- **FRC 的なプラズマ** → 高β
- **コンパクトさ** → トカマクより小さい

### 決定的な違い
- Helion は **燃料問題を先送り** している（³He がない）
- 4サイクル案は **物理的問題を先送り** している（実験データがない）

どちらも「先送り」があるが、燃料がないのは技術では解決できない。
物理は実験で検証できる。この意味で p-B11 路線は原理的に健全。

## 4サイクル案が Helion から学べること

1. **直接発電回路の設計**: 誘導型直接変換の回路設計と効率データ
2. **FRC 衝突合体の物理**: 2つの FRC を衝突させる際の加熱メカニズム
3. **パルス運転のエンジニアリング**: ~1 Hz 繰り返し運転の実装（コイル冷却、排気、電源）
4. **磁気圧縮のダイナミクス**: 外部コイルでプラズマを圧縮する物理
5. **パルスの繰り返し信頼性**: 工学的な繰り返し精度の確保方法

特に **誘導型直接発電** は 4サイクル案にも応用可能。
Zピンチの膨張時に磁束変化で電流を回収する設計は Helion のアプローチと親和性が高い。
