## LINEA のアプローチ: FRC + ミラー磁場

### 3つの柱

```
① FRC（磁場反転配位）
   → プラズマを高β（β≈1）で閉じ込める
   → TAE と同じ基本思想

② ミラー磁場
   → FRC の両端をミラー磁場で塞ぐ
   → 端損失を制御する（= 電子排出の制御に使える）

③ 高エネルギービーム（非熱的核融合）
   → イオンをガモフピーク付近に集中させる
   → マクスウェル分布ではない = 「非熱的」
   → Rider（1995）の前提を崩す発想
```

### なぜこの組み合わせが有力か

```
FRC:
  高β → 磁場コストが小さい → コンパクト
  自分自身の電流で閉じ込め → 外部コイルがシンプル

ミラー磁場:
  FRC の端を磁気ミラーで閉じる
  → 端損失を「制御可能」にする
  → 電子を選択的に端から逃がす調整ができる可能性

高エネルギービーム:
  イオンを特定エネルギー（ガモフピーク）に集中
  → 非マクスウェル分布 → P_fusion が上がる
  → TAE の NBI と同じ方向性だが、「非熱的」を明示的に狙っている
```

## 4サイクル案との類似点（驚くほど多い）

| 項目 | LINEA | 4サイクル案 | 一致度 |
|------|-------|-----------|-------|
| 燃料 | p-B11 | p-B11 | ✅ 完全一致 |
| 制動放射対策 | 非熱的ビーム | 非マクスウェル分布 | ✅ 同じ発想 |
| ミラー磁場の利用 | 端損失制御 | 電子排出（Phase 2b） | ✅ 直結する概念 |
| 閉じ込め | FRC | Zピンチ | ❌ 異なる |
| 運転モード | 準定常（FRC維持） | パルス（ns燃焼） | ❌ 異なる |
| 圧縮トポロジー | S 定常（圧縮しない） | C 体→線→点 | ❌ 異なる |

### 最大の共通点: 「非熱的」× ミラー

LINEA と 4サイクル案は、以下の2点で**全く同じ壁に挑戦している**:

1. **非マクスウェル分布でRiderを回避する**（抜け道①）
2. **ミラー磁場の端損失で電子を制御する**（抜け道③に関連）

これは偶然ではなく、p-B11 の制動放射問題に正面から取り組むと**必然的にここに行き着く**ということ。

### 決定的な違い: 定常 vs パルス

- LINEA（& TAE）: FRC を定常的に維持しながら非熱的状態を保つ
  → τ_eq との永遠の戦い
  → NBI / ビームを打ち続ける必要がある

- 4サイクル案: パルスで燃やして τ_eq より速く逃げる
  → 非熱的状態は ns スケールで崩れても OK
  → ただし繰り返しパルスの工学が必要

## LINEA から学べること

| 項目 | 内容 | 4サイクル案への適用 |
|------|------|------------------|
| **FRC + ミラーの物理** | FRC 端のミラー磁場での粒子挙動データ | Phase 2b の電子排出の設計根拠 |
| **非熱的ビーム核融合** | ガモフピーク集中の実験データ（将来） | 抜け道①の実証可能性評価 |
| **浅井教授の FRC 合体実績** | FRC 生成・維持の具体的な磁場/電流パラメータ | FRC → Zピンチへの知見変換 |
| **坂本教授のミラー実績** | タンデムミラーでの端損失制御 | **電子排出の定量的な実現可能性** |

### 特に注目: CTO 坂本教授のミラー研究

タンデムミラーは「端損失が問題」として1980年代に主流から外れた方式。
しかし LINEA は**端損失を制御可能な機能として再評価**している。

これは我々の「磁気ミラー端損失 = 電子排出」と**全く同じ逆転の発想**。

## 正直な評価

| 項目 | 判定 |
|------|------|
| p-B11 を狙う姿勢 | 🟢 TAE と同じ本気度 |
| FRC + ミラーの組み合わせ | 🟢 物理的に筋が良い |
| 「非熱的」核融合の明示 | 🟢 Rider 回避を意識している |
| 実験実績 | 🟡 FRC 合体は実績あり。p-B11 条件では未実証 |
| 資金力 | 🔴 TAE（~1800億円）に比べて不明（おそらく桁違いに少ない） |
| 定常での非熱的維持 | 🔴 τ_eq の壁は TAE と同じ |

**方向性は正しいが、実験実証と資金が課題。** これは我々（0円の思考実験）にも言えること。
