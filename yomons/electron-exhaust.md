# 電子排出メカニズム — 線の段階で電子を捨てる

> L1-2（制動放射問題）のサブトピック / 圧縮トポロジー ルートC の固有機能
> 関連: [`compression-topology.md`](compression-topology.md) / [`larmor-radius.md`](larmor-radius.md) / [`ti-te-escape.md`](ti-te-escape.md)

---

## 発想

圧縮トポロジーでルート C（体→線→点）を選ぶと、「線」の段階で特別なことができる。
磁場が軸方向に整列した円筒プラズマでは、電子とイオンの運動特性が根本的に異なるため、
**電子だけを選択的に排出**できる。

これは「Ti/Te 分離」の第3の抜け道になりうる。

---

## 線状プラズマにおける電子とイオンの非対称性

### 磁場束縛の差

```
軸方向磁場 B の中で:

電子:
  ラーモア半径 r_L = m_e v / (eB) ~ 0.1 mm
  → 磁力線にガチガチに束縛
  → 軸方向にしか自由に動けない（1D的運動）
  → 熱速度 v_th,e ~ 10⁷ m/s（速い）

イオン（陽子）:
  ラーモア半径 r_L = m_p v / (eB) ~ 4 mm
  → 磁場をまたいで動ける
  → 半径方向にも自由度がある（2D〜3D的運動）
  → 熱速度 v_th,i ~ 10⁵ m/s（遅い）
```

### 軸方向脱出時間の差

プラズマ長を $L \sim 1$ cm として：

$$
\tau_{escape,e} = \frac{L}{v_{th,e}} \approx \frac{10^{-2}}{10^7} = 1 \text{ ns}
$$

$$
\tau_{escape,i} = \frac{L}{v_{th,i}} \approx \frac{10^{-2}}{10^5} = 100 \text{ ns}
$$

**電子は 1 ns で端に到達。イオンは 100 ns かかる。**
→ 端を **数 ns だけ開ける**と、電子は出るがイオンはほぼ出ない。

---

## 3つの電子排出方法

### 方法1: 端開放（磁気ミラー端損失の逆利用）

```
通常の磁気ミラー核融合:
  端からの粒子漏れは「問題」→ 必死に防ぐ

4サイクル案:
  端からの電子漏れを「機能」として利用する

  ←電子流出   [=====線状プラズマ=====]   電子流出→
               (イオンは残留)
```

**動作原理:**
1. Phase 2a（Zピンチ圧縮）で体→線に圧縮。端は磁気ミラーで閉じている
2. Phase 2b で端の磁場を急速に弱める（端を開ける）
3. 電子が軸方向に高速脱出（~1 ns）
4. イオンはまだ動き出していない（~100 ns 必要）
5. 数 ns 後に端を再び閉じる

**所要時間:** 2〜5 ns（電子の数回の往復分）

**メリット:**
- 最もシンプル。追加電極不要
- 磁場コイルの制御だけで実現
- 高エネルギー電子から優先的に逃げる → Te が効率的に下がる

**デメリット:**
- 端を開けるタイミング精度が必要（ns スケール）
- 磁場変化の速度に限界がある

### 方法2: 正電位端電極（バイアス排出）

```
  [+極板]  [=====線状プラズマ=====]  [+極板]
     ↑ 電子を引き寄せ                  ↑
       イオンを押し戻す
```

**動作原理:**
1. 線状プラズマの両端に正バイアス電極を設置
2. 電子（負電荷）は正電極に引き寄せられて回収
3. イオン（正電荷）は正電極に反発して中心に残留

**電圧の目安:**

$$
V_{bias} \sim \frac{k_B T_e}{e} \sim 1\text{〜}10 \text{ kV}
$$

（電子温度 Te ~1〜10 keV に対応する電位）

**メリット:**
- 能動的に電子除去率を制御できる
- ペニングトラップ・イオン源で実績あり
- 端を物理的に開ける必要がない

**デメリット:**
- 電極がプラズマに近い → スパッタ・汚染リスク
- 電流が流れる → 回路設計が必要

### 方法3: ECR選択加熱排出

**動作原理:**
1. 電子サイクロトロン周波数 $f_{ce} = eB/(2\pi m_e)$ にマイクロ波を照射
2. 電子だけが共鳴的にエネルギーを吸収（イオンは共鳴しない）
3. 加速された電子が磁気ミラーの閉じ込め電位を越えて脱出

$$
f_{ce} = \frac{eB}{2\pi m_e} \approx 28 \text{ GHz} \times B[\text{T}]
$$

B = 10 T なら f_ce ≈ 280 GHz（ミリ波領域）

**メリット:**
- 周波数選択性が高い（電子だけに作用、イオンは無関係）
- 非接触（電極汚染なし）
- ECR加熱は既存技術（プラズマエッチング装置等）

**デメリット:**
- GHz帯の高出力マイクロ波源が必要
- ns スケールのパルスでの ECR 加熱は未実証

## デバイ遮蔽による限界

電子を抜くとプラズマが正に帯電し、残った電子を引き戻すアンビポーラ電位が発生する。

$$
\lambda_D = \sqrt{\frac{\varepsilon_0 k_B T_e}{n_e e^2}}
$$

| パラメータ | 値 | λ_D |
|-----------|-----|-----|
| Te = 10 keV, n_e = 10²⁵ m⁻³ | 高温・高密度 | ~0.007 mm |
| Te = 1 keV, n_e = 10²⁴ m⁻³ | 低温・中密度 | ~0.02 mm |

プラズマ半径を r ~ 1 mm とすると：

$$
\frac{r}{\lambda_D} \sim 50\text{〜}150
$$

**完全な電荷分離は不可能だが、表面層（数λ_D ）からの電子除去は可能。**

### 除去可能な電子の割合の概算

表面層からの除去:

$$
\frac{\Delta n_e}{n_e} \sim \frac{\text{表面層体積}}{\text{全体積}} = \frac{2\pi r \cdot \delta \cdot L}{\pi r^2 \cdot L} = \frac{2\delta}{r}
$$

$\delta \sim 数 \lambda_D \sim 0.05$ mm、$r \sim 1$ mm とすると：

$$
\frac{\Delta n_e}{n_e} \sim \frac{2 \times 0.05}{1} = 10\%
$$

### 端方向の排出とアンビポーラ電位の壁

端から見ると、電子は磁力線に沿って体積全体から流出しうる。
しかし、電子が数%抜けた時点でアンビポーラ電位が立ち上がり、残りの電子を強烈に引き戻す。

軸方向のアンビポーラ電位:

$$
\phi_{ambipolar} \sim \frac{k_B T_e}{e} \ln\left(\frac{m_i}{m_e}\right) \sim T_e \times 3.8
$$

（陽子の場合 ln(m_p/m_e) ≈ 7.6、電位は Te の数倍）

**これがプラズマ物理の鉄則: 準中性条件（n_e ≈ n_i）を自動的に維持する。**
磁気ミラー実験で何十年も確認されている事実であり、外部バイアスで簡単には打ち消せない。

外部バイアス電極で打ち消すには Te の数倍（数十kV）が必要だが、
プラズマ中に安定して電場を維持すること自体が非自明。

**現実的に除去できるのは n_e の数%〜せいぜい 20% 程度。1/10 は楽観的すぎる。**

## 4サイクルへの組み込み: Phase 2 の3分割

```
Phase 1:  電子ビーム集束（体 → 準備）        100〜500 ns
Phase 2a: Zピンチ円筒圧縮（体 → 線）         10〜50 ns
Phase 2b: 電子排出（端開放 + 正電位電極）     2〜5 ns    ★新ステップ★
Phase 2c: 軸方向圧縮（線 → 点）              5〜20 ns
Phase 3:  燃焼                                1〜10 ns
Phase 4:  排気                                100〜1000 ns
```

Phase 2b の追加で**合計サイクル時間は 2〜5 ns 増える程度**。
τ_eq（~1 μs）よりはるかに短いので、イオン温度への影響は無視できる。

## Ti/Te 分離への寄与: 現実的な評価

| 抜け道 | メカニズム | 効果 | 実験実証 |
|--------|-----------|------|---------|
| ① 非マクスウェル分布 | 電子ビームでガモフピーク集中 | P_fusion 2〜5倍 | 🔴 未実証 |
| ② 超短パルス Te 抑制 | オーム加熱を時間で切る | Ti/Te 10→50 | 🔴 未実証 |
| **③ 電子排出** | **線段階で端から電子を抜く** | **n_e を 10〜20% 減少** | **🔴 未実証** |

### 現実的な統合効果の概算

制動放射パワー:

$$
P_{brem} \propto Z_{eff}^2 \cdot n_e^2 \cdot T_e^{1/2}
$$

**③の現実的効果:** n_e を 20% 減らせたとして:

$$
P_{brem} \rightarrow (0.8)^2 \times P_{brem} = 0.64 \times P_{brem}
$$

P_fusion/P_brem = 0.22 → 0.34。**これだけでは全然足りない。**

> **※ベースライン更新（2024〜）:** 以下の計算はRider（1995）の0.22を出発点としているが、最新データでは最適条件で P_fus/P_brem ≈ 1.0〜1.2。ただし**真のボトルネックはP_ie（イオン-電子緩和）に移っており**、②超短パルスは計算で否定された（τ_eq/τ_burn=0.0038）。詳細: [`pie-alpha-channeling.md`](pie-alpha-channeling.md) / [`pie-pulse-calculation.md`](pie-pulse-calculation.md)

### 3つ全部組み合わせると

```
① 非マクスウェル分布: P_fusion を 3倍（控えめに見積もり）
② 超短パルス:        Te を 1/5 に → P_brem を 1/√5 ≈ 0.45 倍 ← ※計算で否定済み
③ 電子排出:          n_e を 20% 減 → P_brem を 0.64 倍

総合改善（旧ベースライン基準）:
  P_fusion/P_brem = 0.22 × 3 × (1/0.45) × (1/0.64) ≈ 2.3
```

**NET は超えるが、②が否定されたため①③のみでは不十分。αチャネリング（L1-4）が唯一の生存経路。**

③単体で P_fusion/P_brem を劇的に改善する力はない。
**抜け道③は「②のブースター」程度の位置づけが正直な評価。**

### 致命的な矛盾: 電子を抜くとZピンチできない

Zピンチの電流は**電子が運ぶ**。電子を大幅に減らした状態では：
- Phase 2c（軸方向圧縮）の電流が流れない
- Zピンチ圧縮が成立しない可能性がある

```
電子を抜く → 電流の担い手が減る → Zピンチ弱体化 → 圧縮不足
```

Phase 2a（円筒圧縮）の後に電子を抜くから Phase 2a は問題ないが、
Phase 2c でもう一度圧縮が必要な設計では矛盾が生じる。

**この矛盾が解決しない限り、Phase 2b（電子排出）の有効性は限定的。**

---

## 未解決の課題

1. **🔴 Zピンチとの矛盾**: 電子を抜くと Phase 2c の電流が流れない。これが最大の障壁
2. **アンビポーラ電位の壁**: 準中性条件により除去率は数%〜20%が限界の可能性大
3. **正味正電荷プラズマの安定性**: 電子が少ないプラズマの集団的挙動は未知
4. **端電極の耐久性**: Phase 3 の核融合バースト時に電極が持つか
5. **ns精度の端開閉**: 磁場コイルの応答速度

## 正直な総合評価

| 項目 | 判定 |
|------|------|
| 電子が優先的に端から出る | 🟢 物理的に確実 |
| 数%の電子密度低下は起こる | 🟢 ほぼ確実 |
| n_e を 1/10 にできる | 🔴 アンビポーラ電位で多分無理 |
| 電子排出後にZピンチできる | 🔴 矛盾の可能性あり |
| 単体で制動放射問題を解決 | 🔴 無理 |
| 抜け道①②のブースターとして | 🟡 P_brem を 0.64 倍にする程度 |

**方向性は物理的に正しいが、効果は限定的。「第3の抜け道」というより「②の補助」。**

## なぜルート C でしかできないか

この電子排出メカニズムは**圧縮トポロジー ルート C（体→線→点）の固有機能**。

```
ルート D（体→点）:
  球対称圧縮 → 磁場方向が定まらない → 電子の選択的排出方向がない
  → ❌ 電子排出不可能

ルート C（体→線→点）:
  円筒圧縮 → 磁場が軸方向に整列 → 電子は軸方向にのみ移動
  → ✅ 端から電子を選択的に排出可能
```

**「線にしてから端を開ける」は、ルート C を選んだからこそ可能な操作。**
これが 4サイクル案が球対称爆縮（NIF型）ではなく Zピンチ型を採用すべき最も強い物理的根拠。
