## MagLIF の 3 要素

MagLIF のアプローチは **3 つの要素を組み合わせる** 点が特徴的。

### ① 軸方向磁場（Axial B-field）

```
  ↑ B ↑ B ↑ B ↑ B ↑ B
  ↑   ↑   ↑   ↑   ↑
  │   │   │   │   │
  ┌───────────────┐
  │  D-T 燃料ガス  │  ← ライナー内部
  └───────────────┘
  │   │   │   │   │
  ↑   ↑   ↑   ↑   ↑
```

- 圧縮前にライナー内部に**軸方向の外部磁場**（~10 T）を印加
- 圧縮時に磁束が保存される → 磁場が 10³〜10⁴ T に増幅
- **α粒子のラーモア半径 < 燃料半径** → α粒子が閉じ込められる
- 熱伝導も磁場で抑制 → 温度維持に貢献

### ② レーザー予熱（Laser Preheat）

```
  ~~~~~~~~~~~~~ ← レーザービーム（Z-Beamlet, ~2 kJ）
       ↓
  ┌───────────────┐
  │  D-T 燃料ガス  │  ← 数百 eV に予熱
  └───────────────┘
```

- Z-machine のパルスが到達する**前に**、レーザーで燃料を予熱
- 予熱温度: 数百 eV（~100〜300 eV）
- **なぜ予熱するのか?**
  - 冷たい燃料をそのまま圧縮すると、断熱圧縮効率が悪い
  - 予熱された燃料は圧縮比が小さくても高温に到達できる
  - 必要な圧縮比: ~10（ICF の ~30〜40 と比較して格段に小さい）
- タイミングが極めて重要（レーザー → 数十〜数百 ns 後 → Z-pinch ピーク）

### ③ Z ピンチ・ライナー圧縮

```
  26 MA の電流
       ↓
  ─────→─────→─────→  電流（軸方向）
  ┌─────────────────┐
  │ Be ライナー（金属） │  ← 直径 ~5 mm, 高さ ~7 mm
  └─────────────────┘
       ↑
  J × B 力 → 内向きに圧縮
```

- **ライナー**: ベリリウム（Be）の薄肉円筒（壁厚 ~0.5 mm）
- Z-machine が 26 MA の電流パルスをライナーに流す
- **J × B 力**（ローレンツ力）がライナーを内向きに圧縮
- 圧縮速度: ~70 km/s
- 最終的に燃料が keV 温度・高密度に到達

## 圧縮トポロジー: 体→線→点（C ルート）

```
  [3D 体積]        [1D 線状]          [0D 点]
  ┌─────┐         ┌──┐              ┌┐
  │     │   →→→   │  │    →→→       ││  ← 核融合燃焼
  │     │  J×B圧縮  │  │   更に圧縮    └┘
  │     │         └──┘
  └─────┘
  体（円筒ガス）   線（細い柱）       点（ホットスポット）
```

**これは 4 サイクル案と同じ C ルート。**

- 初期状態: 円筒形の燃料ガス（3D 体積）
- Z ピンチ圧縮: 径方向に圧縮 → 細い線状に（体→線）
- 端部・不均一性: 軸方向にも圧縮効果 → 最終的にホットスポット形成（線→点）
- ICF（レーザー核融合）の「面→点」圧縮とは異なるルート

## 実験実績

### 2014年: 熱核中性子の確認

| 項目 | 数値 |
|------|------|
| 中性子収量 | ~10⁹ 個/ショット |
| イオン温度 | ~3 keV |
| 燃料密度 | ~0.2 g/cm³ |
| 閉じ込め時間 | ~数 ns |

**「熱核中性子が出た」** = 衝突ではなく**熱的な核融合反応**が起きたことの証拠。
これは MagLIF のアプローチが物理的に機能することを示した画期的な成果。

### スケーリング実験（進行中）

- 電流を増加（26 MA → 将来的に 60+ MA の次世代機）
- 中性子収量の電流スケーリング: Y_n ∝ I^α（α ≈ 4〜5 と予測）
- 磁場、レーザーパラメータの最適化が続いている
- 最終目標: 科学的ブレークイーブン

## 4 サイクル案との比較

| 項目 | MagLIF | 4サイクル案 |
|------|--------|-----------|
| 燃料 | **D-T** | **p-B11** |
| 圧縮トポロジー | **C ルート（体→線→点）** | **C ルート（体→線→点）** |
| 圧縮ドライバー | **Z ピンチ** | **Z ピンチ** |
| 磁場 | 軸方向（閉じ込め補助） | あり（閉じ込め補助） |
| 予熱方式 | レーザー | 電子ビーム |
| 運転モード | パルス（~100 ns） | パルス（ns スケール） |
| 制動放射 | 問題にならない（Z_eff 小さい） | **最大の課題**（Z_eff² が大きい） |
| Ti/Te 分離 | 不要（D-T は Ti=Te で動く） | 必須（Ti/Te ≈ 300 必要） |
| 電子ビームフェーズ | **なし** | あり（非マクスウェル分布生成） |
| 電子排出 | **なし** | あり（制動放射抑制のため） |
| 実験実績 | **熱核中性子確認済み** | なし（思考実験） |
| 資金 | DOE 連邦資金（数百億円規模） | 0円 |

### 共通点（本質的に同じもの）

1. **圧縮トポロジーが C ルート**: 体→線→点。円筒形燃料を Z ピンチで圧縮する
2. **Z ピンチをドライバーとして使う**: MA レベルの電流でライナーを内向きに潰す
3. **磁場 + Z ピンチの統合**: 軸方向磁場で閉じ込めを補助する
4. **パルス運転**: 定常ではなく、単発パルスで燃焼させる
5. **予熱の考え方**: 圧縮前に燃料を温めて効率を上げる

### 決定的な違い

**燃料の選択がすべてを変える。**

D-T の場合:
- Z_eff = 1（水素同位体）→ 制動放射は軽い
- Ti = Te でも核融合が成立する（反応率ピーク ~15 keV）
- 電子を気にする必要がほとんどない
- だから MagLIF は **電子ビームフェーズも電子排出も不要**

p-B11 の場合:
- Z_eff = 5（ボロン）→ **Z_eff² = 25 倍の制動放射**
- Ti/Te ≈ 300 が必要（Rider の結論）
- 電子が熱くなると制動放射で即死
- だから 4 サイクル案は **電子ビーム + 電子排出** という追加フェーズが必要

**4 サイクル案は本質的に「MagLIF を p-B11 用に改造したもの」と言える。**
改造の核心は「制動放射問題を解くための電子ビーム前段フォーカシングの追加」。

## 4 サイクル案が MagLIF から学べること

### 1. ライナー内爆の物理

- Be ライナーがどう潰れるか（速度、対称性、残留磁場）
- Rayleigh-Taylor（RT）不安定性の成長率と緩和手法
- ライナー材料の選択基準（密度、強度、電気抵抗率）
- 実験データに基づく流体力学コードの検証

### 2. 磁場 + Z ピンチの統合

- 外部磁場がライナー圧縮にどう影響するか
- 磁束保存の実験的検証（理論値 vs 実測値）
- 磁場が閉じ込めに寄与する条件（β 値の最適化）
- 端部損失の制御方法

### 3. 予熱のタイミングと効果

- レーザー予熱の実験から「いつ、どれだけ予熱するか」の知見
- 予熱温度と最終到達温度の関係（断熱圧縮の効率）
- 4 サイクル案では「電子ビーム」で予熱するが、タイミングの考え方は同じ
- MagLIF の予熱パラメータスタディが直接参考になる

### 4. RT 不安定性の緩和

- Z ピンチ圧縮で最大の敵は **RT 不安定性**
- MagLIF は軸方向磁場で RT を緩和する実験データを持っている
- 磁場強度と RT 成長率の関係 → 4 サイクル案の設計に直接適用可能

## 正直な評価

**MagLIF は D-T では動く。実験で証明された。**

しかし p-B11 に適用しようとすると:
- Z_eff² の壁が立ちはだかる（制動放射が 25 倍）
- 「レーザー予熱 + Z ピンチ圧縮」だけでは Ti/Te 分離ができない
- MagLIF のスキームをそのまま p-B11 に移植しても**制動放射で死ぬ**

4 サイクル案の立場:
- MagLIF の C ルート圧縮 + Z ピンチを**そのまま使う**
- 制動放射問題を解くために**電子ビーム前段フォーカシング**を追加する
- つまり「MagLIF + 電子ビームフェーズ = 4 サイクル案」

これは MagLIF への最大の敬意であると同時に、最大の批判でもある:
**D-T 用に最適化されたスキームを p-B11 に転用するには、根本的な追加要素が必要。**
