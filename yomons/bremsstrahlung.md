> 関連: [`charge-neutrality.md`](charge-neutrality.md) / [`ti-te-escape.md`](ti-te-escape.md) / [`fuel-comparison.md`](fuel-comparison.md)

## 制動放射とは

**電子が加速度運動（方向転換や減速）するとき、電磁波（X線）を放出してエネルギーを失う現象。**

ドイツ語の Bremsstrahlung = brems（ブレーキ）+ strahlung（放射）。直訳で「制動放射」。

### メカニズム

```
電子(-)が飛んでいる
  ↓
イオン(+)の近くを通る
  ↓
クーロン力で軌道が曲がる（加速度が生じる）
  ↓
曲がる瞬間にX線を放出
  ↓
電子のエネルギーが減る = プラズマが冷える
```

**加速度を受ける荷電粒子は電磁波を出す** — これは古典電磁気学の基本法則（ラーモアの公式）。逃げ道がない。

$$
P_{rad} = \frac{q^2 a^2}{6\pi \varepsilon_0 c^3}
$$

- $q$: 電荷
- $a$: 加速度
- $c$: 光速

### 日常の類比

**自転車がカーブを曲がるときのブレーキ音。**

まっすぐ走ってる分には音は出ない。でもカーブで減速・方向転換すると「キーッ」と音が出て、運動エネルギーが音（＝波）に変わって失われる。

電子も同じ。まっすぐ飛んでれば放射しない。イオンの近くで軌道が曲がるとX線を出してエネルギーを失う。

### なぜ電子だけが問題か

イオンも軌道が曲がるが、放射パワーは質量の2乗に反比例する:

$$
P_{rad} \propto \frac{q^2 a^2}{m^2} \propto \frac{1}{m^2}
$$

| 粒子 | 質量 | 放射パワー比 |
|------|------|------------|
| 電子 | $m_e$ | 1（基準） |
| 陽子 | $1836 m_e$ | $1/1836^2 \approx 3 \times 10^{-7}$ |
| B⁵⁺ | $\sim 20000 m_e$ | $\sim 10^{-9}$ |

**イオンの制動放射は電子の 100 万分の 1 以下。事実上ゼロ。**
だから制動放射問題 = 電子の問題。

## 制動放射パワーの公式

$$
P_{brem} = C_B \cdot Z_{eff}^2 \cdot n_e^2 \cdot T_e^{1/2}
$$

| 項 | 意味 | p-B11 での影響 |
|----|------|---------------|
| $Z_{eff}^2$ | イオンの電荷が大きいほど電子を強く曲げる | 18.8（D-Tの18.8倍） |
| $n_e^2$ | 電子が多いほど放射イベントが多い | 6個/反応（D-Tの3倍） |
| $T_e^{1/2}$ | 電子が速いほど放射パワーが大きい | Ti/Te分離で抑えたい項 |

**p-B11 は $Z_{eff}^2$ だけで D-T の 18.8 倍のハンデ。** これが p-B11 核融合の最大の壁。

## 制動放射は回収できないか？ — 3つの方法と致命的な問題

### 方法1: X線を反射して戻す（ターボ的発想）

```
X線が壁に当たる → 壁で反射 → プラズマに戻す → エネルギー回収
```

**致命的な問題が2つ:**

**a) X線は反射できない。**

可視光は鏡で反射できるが、X線は波長が短すぎ（~0.1 nm）て：
- ほぼ全ての物質を透過するか吸収される
- 反射できるのは全反射角（~1°以下）の極浅い角度だけ
- 全方向に飛ぶ制動放射を鏡で集めるのは物理的に無理

**b) 仮に戻せても、電子を加熱してしまう。**

X線を吸収するのは電子（光電効果・コンプトン散乱）。つまり：

```
制動放射（電子のエネルギーがX線として流出）
  ↓ 反射して戻す
電子がX線を吸収（Te が上がる）
  ↓
Te 上昇 → 制動放射がもっと増える → 悪循環
```

**ターボのつもりが暴走エンジンになる。**

### 方法2: プラズマを光学的に厚くする（EGR的発想）

密度を上げまくって、X線がプラズマの外に出る前に再吸収されるようにする。

```
X線が出る → すぐ隣の粒子が吸収 → 外に逃げない → エネルギーが残る
```

**光学的に厚い条件:**

制動放射の自己吸収の平均自由行程:

$$
\lambda_{mfp} \propto \frac{T_e^{7/2}}{n_e^2 Z^2 \nu^3}
$$

核融合条件（Te ~keV, n_e ~10²⁵ m⁻³）では $\lambda_{mfp}$ はプラズマサイズよりはるかに大きい。
→ **プラズマは自分の制動放射に対して透明（光学的に薄い）。X線は素通りする。**

光学的に厚くするには n_e > 10²⁷ m⁻³ 以上が必要（現想定の 100 倍以上）。
そこまで圧縮すると黒体放射が支配的になり、別の膨大な放射損失が発生する。

**しかも再吸収するのは電子。** 方法1と同じ問題で Te が上がるだけ。

### 方法3: 壁で受け止めて熱回収（廃熱回収）

```
X線が壁に当たる → 壁が加熱 → 蒸気タービン → 発電
```

**これは実際にできる。** ただし：

| 項目 | 直接発電（α粒子） | 熱回収（X線→タービン） |
|------|------------------|---------------------|
| 変換効率 | 60〜90%（理論値） | ~40%（カルノー限界） |
| 設備 | 静電/磁気デセレレータ | ボイラー + タービン + 発電機 |
| 中性子遮蔽 | 不要 | 不要 |

p-B11 のメリットは「中性子なし → 直接発電 → 高効率」。
でも制動放射分を熱回収に頼ると、結局タービン発電が必要になり、
**D-T + タービンとほぼ同じ構成になる → p-B11 を選ぶ意味が薄れる。**

## なぜ全部ダメかの本質

### 車のターボ/EGR が機能する理由

```
排気ガス（運動エネルギー） → タービン回転 → 吸気圧縮（有用な仕事）
```

排気エネルギーを **吸気の圧縮（= エンジンの入力側）** に変換できる。
エネルギーの形が変わっても、エンジンの動作に直接貢献する。

### 制動放射で同じことができない理由

核融合で欲しいのは「イオンの加熱」。X線のエネルギーをイオンに渡せれば最高。

```
✗ X線 → イオン加熱:  X線を吸収するのは電子。イオンは吸収しない
                      X線→イオンの変換メカニズムが物理的に存在しない

✗ X線 → 電子加熱:    Te が上がる → 制動放射が増える → 悪循環

△ X線 → 壁加熱 → タービン:  可能だが効率40%。p-B11 の利点が消える
```

**「出さないようにする」（Ti/Te 分離）が唯一の戦略。「出たものを回収する」は機能しない。**

これが制動放射問題の本質的な厄介さであり、Ti/Te 分離・非マクスウェル分布・超短パルスなどの「出さない」系の抜け道に全てを賭けている理由。

## まとめ: 制動放射の全体像

```
制動放射とは:
  電子が曲がるとX線が出てエネルギーが逃げる

なぜ p-B11 で深刻か:
  Z_eff² = 18.8 → D-T の 18.8 倍のハンデ
  さらに電子が 6個/反応（D-T の 3 倍）

回収できるか:
  ✗ 反射して戻す → X線は反射できない & 電子が加熱されて悪循環
  ✗ プラズマで自己吸収 → 透明すぎて素通り & 電子加熱
  △ 壁で熱回収 → 可能だが効率40%。p-B11の利点消滅

唯一の対策:
  「出さないようにする」= Ti/Te 分離
  → 非マクスウェル分布 + 超短パルス + 電子排出（補助）
```
