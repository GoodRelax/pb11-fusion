## TAE のアプローチ: FRC + NBI

### FRC（Field-Reversed Configuration）とは

通常の磁場閉じ込め（トカマク）では外部磁場がプラズマを閉じ込める。
FRC では**プラズマ自身が作る電流で磁場を反転させ、自分自身を閉じ込める**。

```
トカマク:
  外部コイルの磁場 → → → → → →
  プラズマ         → → → → → →  （同じ方向）

FRC:
  外部磁場         → → → → → →
  プラズマ内部磁場   ← ← ← ← ← ←  （逆方向！）
  　　　　　　　　　↑
  　　分離面（separatrix）で磁場がゼロになる
```

**メリット:**
- β値（プラズマ圧/磁場圧）が高い → 磁場が小さくて済む → コンパクト
- 分離面の内側は磁場ゼロに近い → β ≈ 1（プラズマ圧が支配）
- 形がシンプル（ラグビーボール型）

**デメリット:**
- 不安定（ティルト不安定性、回転不安定性）
- FRC を長時間維持するのが難しい

### NBI（中性粒子ビーム入射）段階加熱

TAE の最大の武器。**高速中性粒子ビーム（NBI）でイオンだけを選択的に加熱する。**

```
NBI の仕組み:
  ① 水素イオンを加速（数十keV）
  ② 中性化セル（ガス）を通して電子をくっつけ、中性粒子にする
  ③ 中性粒子は磁場を無視してプラズマ中心まで到達
  ④ プラズマ内で電離 → 高速イオンとして捕獲
  ⑤ イオンだけにエネルギーが入る → Ti が上がる
```

**結果: Ti > Te が実現する。** これが TAE の Ti/Te 分離の方法。

## C-2W "Norman" の実績

TAE の5代目装置（2017〜）で達成したこと:

| 項目 | 数値 |
|------|------|
| イオン温度 Ti | ~2000 eV（2 keV） |
| 電子温度 Te | ~500 eV（0.5 keV） |
| **Ti/Te 比** | **~4** |
| FRC 寿命 | ~30 ms（ミリ秒） |
| プラズマ密度 | ~10¹⁹ m⁻³ |

**Ti/Te ≈ 4 を実験で実証。** これは小さい数字に見えるが、
「Ti > Te を意図的に作れる」ことを示した唯一の核融合実験。

### ただし p-B11 には遠い

| 必要値（p-B11） | C-2W 実績 | ギャップ |
|----------------|----------|---------|
| Ti ~ 300 keV | 2 keV | 150倍 |
| Ti/Te ≈ 300（Rider） | 4 | 75倍 |
| n ~ 10²⁵ m⁻³ | 10¹⁹ m⁻³ | 100万倍 |

**まだ数桁足りない。** しかし「方向性は正しい」ことを示した意義は大きい。

## TAE のロードマップ

```
C-2W "Norman" (2017-)     : FRC + NBI で Ti/Te > 1 を実証 ← 今ここ
  ↓
"Copernicus" (建設中, ~2025) : 温度を 10x に引き上げ。D-T 等価条件に接近
  ↓
"Da Vinci" (計画)          : D-T で Q > 1 を実証
  ↓
最終目標                   : p-B11 で発電実証
```

TAE は「まず D-T で技術を実証してから p-B11 に移行する」段階的アプローチ。
p-B11 到達は 2030 年代後半〜2040 年代の見込み。

## TAE vs 4サイクル案: 類比と対比

| 項目 | TAE | 4サイクル案 |
|------|-----|-----------|
| 最終燃料 | p-B11 | p-B11 |
| 閉じ込め | FRC（磁気） | Zピンチ（ハイブリッド） |
| 運転モード | 準定常（FRC を維持し続ける） | パルス（ns スケールで燃焼→リセット） |
| Ti/Te 分離の方法 | NBI 段階加熱（イオンだけに入れる） | 非マクスウェル分布 + 超短パルス + 電子排出 |
| Ti/Te 実績 | **4（実験実証あり）** | なし（思考実験） |
| 制動放射対策 | Ti/Te 分離で抑制 | Ti/Te 分離 + パルスで逃げる |
| 発電方式 | 直接変換（目標） | 直接変換 |
| 圧縮トポロジー | S 定常（圧縮しない） | C 体→線→点 |
| 密度 | 低い（10¹⁹ m⁻³） | 超高密度を目指す（10²⁵ m⁻³） |
| 資金 | ~1800億円 | 0円 |

### 共通点
- **p-B11 を本気で狙っている**
- **Ti/Te 分離が鍵だと理解している**
- **直接発電を前提にしている**

### 決定的な違い
- TAE は**定常運転**で Ti/Te 分離を維持しようとする → τ_eq（~1μs）との永遠の戦い
- 4サイクル案は**パルスで逃げる** → τ_burn < τ_eq なら勝ち

TAE の課題: Ti/Te = 4 を **300 まで上げる** のは定常運転では極めて困難。
NBI のパワーを上げれば Ti は上がるが、いずれ電子にもエネルギーが移る（τ_eq の壁）。

4サイクル案の利点: パルスなら τ_eq に勝てる可能性がある。
4サイクル案の弱点: 実験データがゼロ。

## 4サイクル案が TAE から学べること

1. **NBI 段階加熱の物理データ**: Ti/Te 比がどう決まるかの実験値
2. **FRC プラズマの安定性**: ティルト不安定性の制御方法
3. **直接発電の回路設計**: α粒子からの直接エネルギー回収
4. **段階的アプローチの戦略**: まず D-T で技術実証 → p-B11 へスケールアップ
