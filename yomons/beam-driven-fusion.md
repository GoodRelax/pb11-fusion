# ビーム駆動非熱核融合 — プラズマを燃やさずにビームで反応させる

> **一言で:** プラズマ全体を数十億度に加熱する代わりに、高エネルギービーム（数百keV）をプラズマに打ち込んで直接核融合させる。制動放射問題を「構造的に回避」する可能性がある——が、電子阻止能という別の壁にぶつかる。

---

## 熱核融合 vs ビーム駆動非熱核融合

### 熱核融合（従来のアプローチ）

```
燃料全体を加熱 → 全粒子が高温 → マクスウェル分布の高エネルギー端で反応
  ↓
Ti = Te = 300 keV（全体が30億℃）
  ↓
電子も300 keV → P_brem ∝ Te^(1/2) が巨大
  ↓
P_fus/P_brem ≈ 1.0（ギリギリ）+ P_ie問題
```

**問題:** プラズマ全体を加熱すると電子も温まり、制動放射でエネルギーが逃げる。

### ビーム駆動非熱核融合

```
プラズマは比較的低温に保つ（Te ~ 数keV）
  +
高エネルギー陽子ビーム（~600 keV）を入射
  ↓
ビームイオンが直接ホウ素と反応
  ↓
バルクプラズマは低温のまま → P_brem が小さい
```

**利点:** プラズマ全体を加熱しないので、電子温度が低く保たれ、制動放射が抑制される。

---

## 3つの「ビーム核融合」の違い

| 方式 | ターゲット | ビーム | 電子の状態 | 実例 |
|------|-----------|--------|-----------|------|
| **ビーム-固体** | 固体ホウ素 | 陽子ビーム | 束縛電子（阻止能） | HB11レーザー |
| **ビーム-プラズマ** | ホウ素プラズマ | 陽子ビーム | 自由電子（クーロン減速） | LINEA, TAE |
| **ビーム-ビーム** | 陽子ビーム | ホウ素ビーム | なし（理想）/ 電子雲あり | CBFR概念 |

本ドキュメントは**ビーム-プラズマ方式**を扱う。ビーム-固体は [beam-target.md](beam-target.md) を参照。

## ビーム-プラズマ方式の物理

### ビームイオンの減速過程

高エネルギー陽子（~600 keV）がプラズマに入射されると、3つの過程で減速する：

```
ビーム陽子（600 keV）
  │
  ├── 電子との衝突で減速（クーロン散乱）  ← エネルギーの大部分がここに消える
  │     → 電子が加熱される → 制動放射
  │
  ├── イオンとの衝突で散乱・減速
  │     → イオンが加熱される（有益）
  │
  └── ホウ素イオンとの核融合反応（確率 << 1）
        → 3α + 8.7 MeV（目標！）
```

### 臨界速度（v_c）

ビームイオンの速度がある臨界値 v_c を超えると、**電子への減速が支配的**になる：

$$
v_c = \left(\frac{3\sqrt{\pi}}{4}\right)^{1/3} \left(\frac{m_e}{m_i}\right)^{1/3} v_{te} \approx 0.1 \times v_{te} \times (m_i/m_e)^{1/3}
$$

p-B11の条件では：
- ビーム陽子速度（600 keV）: v_beam ≈ 1.1 × 10⁷ m/s
- 臨界速度（Te = 10 keV）: v_c ≈ 4 × 10⁶ m/s

**v_beam > v_c なので、ビームエネルギーの大部分は電子に渡る。**

### 電子へのエネルギー分配比

ビームイオンが完全に減速するまでに、電子とイオンに渡るエネルギーの比率：

| Te (keV) | 電子に渡る割合 | イオンに渡る割合 | 核融合する割合 |
|----------|---------------|----------------|--------------|
| 1 | ~95% | ~5% | ~0.01% |
| 10 | ~80% | ~20% | ~0.1% |
| 50 | ~60% | ~40% | ~0.5% |
| 100 | ~50% | ~50% | ~1% |

**電子温度が低いほど電子への分配が多い** — 電子を冷やそうとするほど、ビームエネルギーが電子に吸い取られるというジレンマ。

## P_ie問題との関係

ビーム駆動方式は「P_ie問題を回避している」ように見えるが、**形を変えているだけ** の可能性がある：

### 熱核融合のP_ie問題

```
バルクイオン（300 keV）→ 電子に緩和（P_ie）→ 制動放射（P_brem）
```

### ビーム駆動のP_ie問題（変形版）

```
ビーム陽子（600 keV）→ 電子に減速（ビーム阻止能）→ 制動放射（P_brem）
```

**本質は同じ：** 高エネルギーイオンのエネルギーがクーロン衝突で電子に渡り、制動放射として失われる。

### 定量比較

| パラメータ | 熱核融合 | ビーム駆動 |
|-----------|---------|-----------|
| エネルギー源 | バルクイオン温度 | ビームイオン運動エネルギー |
| 電子への移行メカニズム | イオン-電子緩和（P_ie） | ビーム-電子阻止能 |
| 特性時間比 | τ_eq/τ_burn = 0.0038 | τ_slow/τ_fus ~ 同程度 |
| 密度依存性 | 密度非依存 | 密度非依存（同じ理由） |

ビーム減速時間 τ_slow と核融合反応時間 τ_fus の比も、熱核融合の τ_eq/τ_burn と同様に**クーロン断面積 >> 核反応断面積**で決まり、密度に依存しない。

## それでもビーム駆動が有利な点

### 1. 非マクスウェル分布の維持

熱核融合ではクーロン衝突がイオンをマクスウェル分布に引き戻す（Riderの議論）。
ビーム駆動では**ビーム自体が非マクスウェル分布**であり、維持コストはビーム入射パワーそのもの。

```
熱核融合:  マクスウェル分布を崩すのにエネルギーが必要（Rider: P_recir > P_fus）
ビーム駆動: ビーム入射 = 非マクスウェル分布の維持（ビーム電力がそのまま維持コスト）
```

### 2. 反応エネルギーの制御

ビームエネルギーを ~620 keV（共鳴ピーク）に合わせられる。マクスウェル分布ではイオンの大部分が共鳴から外れるが、ビームでは**全イオンが共鳴エネルギー**。

反応率の改善：
- マクスウェル（Ti=300 keV）: <σv> ≈ 2×10⁻²² m³/s
- ビーム（E=620 keV、ターゲット静止）: σ × v_beam ≈ 1.2×10⁻²⁸ × 1.1×10⁷ ≈ 1.3×10⁻²¹ m³/s

ビームの方が**~6倍高い反応率**。ただしこれは1粒子あたりの反応率であり、ビーム電流（密度）が低いと全体の出力は小さくなる。

### 3. ミラー端損失による自然な電子排出（LINEA方式）

FRC + タンデムミラーの開放端から**軽い電子が先に逃げる**。
閉じ込め領域でTi > Teの非平衡が自然に形成される。

```
ミラー端での損失率:
  電子: 速い（v_e >> v_i）→ 先に逃げる
  イオン: 遅い → 閉じ込められる
  → 自然にTi/Te > 1が形成
  → ただしアンビポーラ電位が電子の逃げを抑制（上限あり）
```

## エネルギー収支の概算

### ビーム駆動p-B11の概算

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| ビーム陽子エネルギー | 600 keV | 共鳴ピーク |
| 核融合確率（1回の通過） | ~10⁻³〜10⁻² | プラズマ密度・厚さに依存 |
| 核融合エネルギー | 8.7 MeV | p + ¹¹B → 3α |
| ビーム→電子損失 | ~80% | Te=10 keV の場合 |
| ビーム→イオン | ~20% | |
| ビーム効率（壁コンセント） | ~50%（NBI） | 中性子ビーム入射 |

**Q 値の概算:**

$$
Q = \frac{P_{fus}}{P_{beam}} = \frac{n_B \cdot \sigma v \cdot E_{fus}}{E_{beam} / \tau_{slow}} \times \eta_{recov}
$$

ビーム1個あたり：
- 入力: 600 keV ÷ 壁コンセント効率50% = 1.2 MeV
- 核融合確率: ~10⁻³（楽観）
- 出力: 10⁻³ × 8.7 MeV × 直接変換70% = 0.006 MeV

**Q ≈ 0.006 / 1.2 ≈ 0.005** — beam-target（固体）と同程度。

### なぜこんなに低いか

核融合確率が低すぎる。ビーム陽子は核融合する前に**電子に減速されて止まる**。

```
ビーム陽子の一生:
  入射（600 keV）
    ↓ 電子との衝突で減速
    ↓ 500 keV... 400 keV... 300 keV...
    ↓ 核融合共鳴ウィンドウを通過（確率~10⁻³）
    ↓ 200 keV... 100 keV...
    ↓ 熱化してバルクイオンに混ざる
  停止

  → 1000個打ち込んで1回核融合すれば良い方
```

## ビーム-固体 vs ビーム-プラズマ

| | ビーム-固体 | ビーム-プラズマ |
|---|---|---|
| ターゲット | 固体ホウ素 | ホウ素プラズマ |
| 電子の状態 | 束縛電子 | 自由電子 |
| 阻止能 | ~100 keV/μm（高い） | 密度依存（低密度なら低い） |
| ビーム飛程 | ~6 μm | mm〜cm（低密度プラズマ） |
| 核融合確率 | ~10⁻⁶ | ~10⁻³〜10⁻² |
| Q値 | ~10⁻⁵ | ~10⁻³〜10⁻² |
| 利点 | 装置が簡単 | 直接変換が可能、反応率が高い |

**ビーム-プラズマの方が100〜1000倍マシ** だが、Q > 1 にはまだ2〜3桁不足。

## ビーム駆動方式を採用しているプロジェクト

| プロジェクト | ビーム方式 | 閉じ込め | 状態 |
|------------|-----------|---------|------|
| TAE Technologies | NBI（中性ビーム入射） | FRC | 🟡 C-2Wで実験中 |
| LINEA Innovations | 水素ビーム | FRC + タンデムミラー | 🔴 初期段階 |
| LPPFusion | DPFピンチ（自己ビーム） | Dense Plasma Focus | 🟡 イオン温度達成 |

### TAEの戦略

TAEは「純粋なビーム駆動」ではなく**ビーム+バルク加熱のハイブリッド**：
1. NBIでFRCプラズマを加熱・安定化
2. ビームイオンが直接核融合する成分 + バルクプラズマの熱核融合成分の両方を活用
3. 将来的にαチャネリングでエネルギー回収を改善

### LINEAの戦略

「非熱核融合」を全面に打ち出し、**バルク加熱を必要としない**と主張：
1. 高エネルギービーム（数百keV）をホウ素プラズマに入射
2. ビームイオンが直接反応
3. ミラー端損失で電子を自然排出
4. 直接エネルギー変換

**正直な評価:** 原理は明確だが、上述のQ値問題（ビーム→電子損失が支配的）への定量的回答が未公開。

## 結論

ビーム駆動非熱核融合は制動放射問題を「構造的に回避」するように見えるが：

1. **ビーム→電子阻止能** がP_ieの代わりに立ちはだかる
2. **核融合確率** が低すぎて Q << 1（現状推定 ~10⁻³〜10⁻²）
3. **クーロン断面積 >> 核反応断面積** という根本的事実は変わらない

```
熱核融合:   イオン温度 → 電子に緩和 → 制動放射  （P_ie問題）
ビーム駆動: ビーム     → 電子に減速 → 制動放射  （阻止能問題）

形が違うだけで、本質は同じ: 電子がエネルギーを奪う
```

**ビーム駆動方式がQ > 1を達成するには、αチャネリング（ビーム版：α粒子→波動→ビームイオン再加速）またはマルチパス方式（ビームを何度もプラズマに通す）が必要。**
