# 圧縮トポロジー — 体から点への4つのルート

> L1（物理層）/ L2（工学層）横断トピック
> 関連: [`L1-physics.md`](L1-physics.md) / [`two-temp-plasma.md`](two-temp-plasma.md) / [`ti-te-escape.md`](ti-te-escape.md)

---

## 発想

核融合で「プラズマを点に圧縮する」とき、3D（体積）→ 0D（点）への道は一通りではない。
中間次元 {2D（面）, 1D（線）} を経由するかどうかで **4つのルート** が存在する。

$$
\text{ルート数} = 2^{|\{2D, 1D\}|} = 2^2 = 4
$$

---

## 4つの圧縮ルート

### ルート A: 体→面→線→点（段階圧縮）

```
[3D 体積] → 平板化 → [2D 面] → 絞り → [1D 線] → ピンチ → [0D 点]
```

- **幾何学**: 最も段階的。各ステップで1次元ずつ潰す
- **メリット**: 各段階で独立に最適化できる。不安定性を段階ごとに制御可能
- **デメリット**: 時間がかかる。フェーズ数が多い
- **実例**: 多段ライナー圧縮（概念段階）

### ルート B: 体→面→点（平面収束）

```
[3D 体積] → 平板化 → [2D 面] → 収束 → [0D 点]
```

- **幾何学**: まず片軸方向に圧縮して薄い円盤にし、そこから中心に収束
- **メリット**: 平面内の圧縮は対称性が高い
- **デメリット**: 面→点の収束が難しい（2次元分を一気に潰す）
- **実例**: 一部のレーザー駆動方式（ホーロラウム内の平板ターゲット）

### ルート C: 体→線→点（Zピンチ型） ← 4サイクル案

```
[3D 体積] → 円筒圧縮 → [1D 線] → 軸方向ピンチ → [0D 点]
```

- **幾何学**: まず半径方向に圧縮して細い柱にし、次に軸方向に縮める
- **メリット**:
  - Zピンチの自然な圧縮方向（半径方向）を活かせる
  - **磁場が軸方向 → 電子をラーモア半径で軸方向に閉じ込め/排除しやすい**
  - Ti/Te 分離と相性が良い
- **デメリット**: ソーセージ不安定性・キンク不安定性が支配的
- **実例**: **Zピンチ、MagLIF、4サイクル案**

### ルート D: 体→点（球対称爆縮）

```
[3D 体積] → 球対称圧縮 → [0D 点]
```

- **幾何学**: 全方向から均等に圧縮。最も直接的
- **メリット**: 圧縮効率が最も高い（全方向からエネルギーを集中）
- **デメリット**:
  - 完全球対称が必要 → レイリー・テイラー不安定性に極めて敏感
  - 磁場の方向性がない → **電子排除（Ti/Te 分離）が困難**
  - 電子もイオンも均等に圧縮される → Te が上がりやすい
- **実例**: NIF（国立点火施設）、レーザー慣性閉じ込め

---

## 各ルートと Ti/Te 分離の相性

| ルート | 圧縮の対称性 | 磁場方向の自由度 | 電子排除のしやすさ | Ti/Te 分離 |
|--------|------------|----------------|------------------|-----------|
| A: 体→面→線→点 | 低い（段階的） | 各段階で設定可 | 🟡 中程度 | 🟡 |
| B: 体→面→点 | 中程度 | 面内に限定 | 🟡 中程度 | 🟡 |
| **C: 体→線→点** | **円筒対称** | **軸方向に明確** | **🟢 最も容易** | **🟢** |
| D: 体→点 | 球対称 | 方向性なし | 🔴 困難 | 🔴 |

**ルート C（Zピンチ型）が Ti/Te 分離に最も有利。**

理由：
1. 円筒圧縮で磁場が軸方向に整列する
2. 電子のラーモア半径（~0.1 mm）が小さいため、磁場に強く束縛される
3. イオンのラーモア半径（~数 mm〜cm）は大きいため、磁場を横切って圧縮される
4. この差を使って「イオンだけ圧縮、電子は磁場に沿って排除」が可能

---

## 各ルートと不安定性の関係

| ルート | 支配的な不安定性 | 深刻度 |
|--------|----------------|--------|
| A: 体→面→線→点 | 各段階で異なる（RT + ソーセージ + キンク） | 🟡 分散 |
| B: 体→面→点 | RT不安定性（面の変形） | 🔴 高い |
| C: 体→線→点 | ソーセージ不安定性（線の太さの揺らぎ）、キンク不安定性（線の曲がり） | 🟡 中程度 |
| D: 体→点 | RT不安定性（球面の変形）が極めて深刻 | 🔴 最も高い |

### ルート C の不安定性対策

```
ソーセージ不安定性: 線の一部が太く/細くなる
→ 対策: 軸方向磁場で安定化（MagLIF方式）

キンク不安定性: 線全体がくねる
→ 対策: 外部磁場 + 壁安定化
```

---

## 4サイクル案がルート C を選ぶ物理的必然性

```
4サイクル案の要求:
  ① Ti/Te 分離が必要（制動放射抑制のため）
  ② パルス式（ns スケールで燃焼を終える）
  ③ 電子ビーム + Zピンチの組み合わせ

       ↓

ルート C（体→線→点）が唯一の自然な選択:
  ① 円筒磁場で電子排除 → Ti/Te 分離に最適
  ② Zピンチは本質的に円筒圧縮 → ルート C そのもの
  ③ 電子ビームは軸方向に入射 → 円筒の軸と一致
```

逆に言えば、**ルート D（球対称爆縮）を選んだ瞬間、Ti/Te 分離が極めて困難になる。**
これが NIF 型アプローチが p-B11 に向かない根本的な理由の一つ。
