# イオンと電子の磁場応答 — ラーモア半径と電子排除

## 用語の定義

### ローレンツ力

磁場中を動く荷電粒子には力が働く。

$$
\vec{F} = q\vec{v} \times \vec{B}
$$

電子もイオンも、電荷を持つので**両方とも**磁場に反応する。

### ラーモア半径（サイクロトロン半径）

磁場中で荷電粒子が描く円軌道の半径。

$$
r_L = \frac{mv}{qB}
$$

| 変数 | 意味 |
|------|------|
| m | 粒子の質量 |
| v | 速さ |
| q | 電荷量 |
| B | 磁場の強さ |

**質量が大きいほど半径が大きい。**

---

## 電子とイオンの軌道の違い

同じ温度・同じ磁場のとき：

| 粒子 | 質量比（電子=1） | ラーモア半径 | 回転方向 |
|------|--------------|------------|---------|
| 電子 | 1 | ~0.1 mm | 右回り |
| 陽子（p） | 1836 | ~4 mm（43倍） | 左回り |
| B11イオン | 約20,000 | ~数 cm（さらに大） | 左回り |

回転方向は電荷の符号で逆になる（電子はマイナス、イオンはプラス）。

## 「電子だけ吹き飛ばす」は原理的にできる

### 磁気フィルター（磁気シース）

```
磁場の強さを適切に設定する
　↓
電子（小さい軌道）→ 磁場に強く曲げられて壁に誘導される
イオン（大きい軌道）→ 磁場をほぼ無視して直進する
　↓
電子だけ選択的に排除できる
```

**実際に使われている技術：**
- ホールスラスター（宇宙推進エンジン）
- 半導体製造のプラズマエッチング装置

### 電場によるポテンシャル障壁

```
負の電圧をかけた壁
→ 電子（マイナス電荷）は反発して戻る
→ イオン（プラス電荷）は引き寄せられる
```

### 電子サイクロトロン共鳴（ECR）

```
電子のサイクロトロン周波数 ～ GHz
イオンのサイクロトロン周波数 ～ MHz
　↓
GHz の電磁波を当てると電子だけを選択的に加熱・排除できる
```

## なぜ「全排除」は無理か：デバイ遮蔽

### 問題1：電気的中性の壁

```
電子を抜く
　↓
プラズマ全体がプラスに帯電する
　↓
強烈な引力が発生して電子が引き戻される
　↓
外から電子が流れ込んでくる
```

これを**デバイ遮蔽**という。プラズマは常に電気的中性（電荷ゼロ）に戻ろうとする性質がある。

デバイ長（中性化が維持される距離）：

$$
\lambda_D = \sqrt{\frac{\epsilon_0 k T_e}{n_e e^2}}
$$

この長さより長い距離で電荷の偏りを維持することはできない。

### 問題2：電子がないと閉じ込めが崩壊する

```
磁気閉じ込めはイオンと電子のセットで成立している
電子を全部抜くと閉じ込めの構造自体が崩壊する
```

### 問題3：排除コストが核融合を上回る

電子を排除し続けるために必要なエネルギーが、
得られる核融合エネルギーより大きくなる可能性がある。

## 現実的な落とし所

「全排除」は不可能だが**「減らす・冷やす」**は有効な戦略。

| アプローチ | 効果 | 実現性 |
|-----------|------|--------|
| ECR冷却（電子サイクロトロン共鳴） | Te を選択的に下げる | 🟡 研究段階 |
| 磁気シースで電子を壁に誘導 | 電子密度を局所的に下げる | 🟡 部分実証 |
| Phase 1 の電子ビームを低エネルギーに抑える | 電子加熱を最小化 | 🟢 設計で対応可能 |

---

## 4サイクル方式への示唆

```
Phase 1（電子ビーム）:
  電子ビームのエネルギーを低く抑える
  → イオンを「引き寄せる」だけで「加熱しない」設計
  → Te を上げずに済む

Phase 2（Zピンチ）:
  磁場シースを炉壁付近に設計
  → 圧縮中に発生した余分な電子を選択的に壁へ誘導

Phase 4（排気）:
  コレクターリングに負電圧をかける
  → α粒子（プラス）を回収しつつ電子を反発
  → 直接発電と電子排除を同時に行う
```

この構造は**4サイクルのフェーズ分けと磁気フィルター原理が自然に組み合わさる**点で有望。
