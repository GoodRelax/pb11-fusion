# CBFR（衝突ビーム核融合炉）— TAEの出発点と、なぜ断念されたか

> **一言で:** 陽子ビームとホウ素ビームを正面衝突させて核融合する。理想的に見えたが、Riderの定量的反論とTAE自身の実験で方針転換。提唱者が自分で見切りをつけた方式。

---

## 1. CBFRとは

1997年にRostoker, Binderbauer, Monkhorst がScience誌に発表（Science 278, 1419, 1997）。TAE Technologies（旧Tri Alpha Energy）はこの論文から創業した。

### 基本構造

```
     陽子ビーム →            ← ホウ素ビーム
     (~600 keV)              (~600 keV相当)
         ↓                       ↓
    ┌───────────────────────────────────┐
    │      FRC（磁場反転配位）            │
    │                                   │
    │   陽子 → → →   ← ← ← ホウ素      │
    │     反対方向に回転する2つのビーム    │
    │                                   │
    │   大ラーモア半径軌道で安定閉じ込め   │
    └───────────────────────────────────┘
                 ↓
         衝突エネルギー ~620 keV
         = 共鳴ピーク（反応率最大）
```

FRC内部で2つのビームが反対方向に回転し、すれ違うたびに衝突して核融合する。加速器の衝突実験と同じ発想だが、ビームをリングに閉じ込めて何度もすれ違わせる。

### なぜ理想的に見えたか

| 利点 | 理由 |
|------|------|
| **非マクスウェル分布が自然に実現** | ビーム自体がピーク型のエネルギー分布 |
| **共鳴エネルギーに最適化可能** | ビームエネルギーを~620 keVに合わせれば反応率最大 |
| **Ti >> Te が構造的に成立** | ビームイオンは高エネルギー、電子は低温で中和のみ |
| **FRCの高β閉じ込め** | 大ラーモア半径軌道がFRCと相性が良い |

Rostokerの論文では「p-B11核融合の実現可能な道」として提案された。

## 2. 3つの壁

### 壁1: 電子は消えない

2本の正イオンビームは互いにクーロン反発する。空間電荷中和のために電子が**必須**：

$$
n_e = n_p + Z_B \cdot n_B = n_p + 5 n_B
$$

等モル混合なら n_e = 6 n_0。**ビーム-プラズマと同じ数の電子が存在する。**

電子は低温（数keV）に保てるはずだが：
- ビームイオンとのクーロン衝突で電子が加熱される（ドラッグ力）
- 加熱された電子が制動放射を出す
- **P_brem の問題は形を変えて残る**

### 壁2: ビームの熱化（Riderの反論）

**Rider, Physics of Plasmas 4, 1039 (1997)** — CBFRの定量的否定：

#### 問題の本質

ビームイオン同士がクーロン衝突すると、速度分布が散乱される。ビームのピーク型分布がマクスウェル分布に「ぼやける」：

```
初期（ビーム分布）:          時間経過後（マクスウェル化）:

  ↑ f(E)                      ↑ f(E)
  │  █                        │
  │  █                        │   ██
  │  █                        │  ████
  │  █                        │ ██████
  └──────→ E                   └──────→ E
     共鳴ピーク                  ← 広がって共鳴から外れる
```

#### 定量評価

| パラメータ | 値 |
|-----------|-----|
| ビーム-ビーム衝突による散乱時間 | τ_scatter |
| 核融合反応時間 | τ_fusion |
| **τ_scatter / τ_fusion** | **<< 1**（散乱が数桁速い） |
| ビーム分布維持に必要な再循環パワー | P_recir |
| **P_recir / P_fus** | **≈ 0.49** |

P_recir/P_fus ≈ 0.49 は「核融合出力の約半分をビーム維持に使う」ことを意味する。

**これだけなら致命的ではない**（電力の半分を再循環に回す発電所は工学的に可能）。しかし**電子へのドラッグ損失を加えると赤字に転落する。**

#### Riderの結論

> ビーム分布の維持コスト（エントロピー生成率から導出）が核融合出力を超える。
> CBFRでの正味エネルギー利得は不可能。

#### Rostokerの反論（Science 278, 1419, 1997）

- FRC中の大ラーモア半径軌道イオンは古典的二体衝突緩和より遅い緩和の可能性
- 波動-粒子相互作用による準安定分布の維持
- Riderは均一プラズマを仮定しており、FRCの空間非一様性を無視

#### Nevinsの独立検証（DTIC報告, 1998）

> Riderの議論を独立に支持。CBFRでの正味エネルギー利得は困難。

**反論は定量的にRiderを覆すに至っていない。**

### 壁3: ルミノシティ問題

加速器物理の衝突実験と同じ問題。核融合反応率は：

$$
R = n_1 \cdot n_2 \cdot \langle\sigma v\rangle \cdot V
$$

十分な反応率を得るにはビーム密度 n₁, n₂ を上げる必要がある。しかし密度を上げると：

- 空間電荷効果が増大 → ビームが発散
- ビーム内散乱（IBS: Intra-Beam Scattering）が増大 → 熱化が加速
- より多くの電子が中和に必要 → 電子への損失が増大

```
密度を上げる → 反応率が上がる → 良い！
  だが同時に
密度を上げる → 散乱が速くなる → ビーム維持コスト増大 → 悪い！

→ どちらも n² に比例するため、密度を上げてもP_recir/P_fusは改善しない
  （P_ie/P_fusの密度非依存性と同じ物理！）
```

## 3. TAEの方針転換

```
1997: CBFR（ビーム-ビーム衝突核融合）でTri Alpha Energy創業
  │   Rostoker, Binderbauer, Monkhorst (Science 278, 1419)
  │
  ↓  Riderの反論（1997）+ Nevinsの独立検証（1998）
  ↓  + 実験的困難（ビーム閉じ込めの不安定性）
  │
2000年代: FRC + NBI（ビーム-プラズマのハイブリッド）に転換
  │   ビーム-ビームの衝突ではなく、
  │   NBIでFRCプラズマを加熱・安定化する方式に変更
  │
2011-2014: C-2 — FRC安定化を実証
  │
2014-2017: C-2U — NBIによるFRC寿命延長
  │
2017-現在: C-2W — Ti+Te > 3 keV、安定30ms閉じ込め
  │
2023: LHDでp-B11核融合の初計測（Nature Comm.）
  │
2025: NBIのみでFRC生成に成功（Nature Comm.）
  │
将来: Da Vinci → αチャネリング導入？
```

**提唱者（Rostoker/Binderbauer）自身がCBFRからFRC+NBIに方針転換した。** これが最も雄弁な評価。

## 4. CBFRの教訓

### beam-target / beam-plasma / beam-beam の比較

| 方式 | ターゲット | 電子 | 核融合確率 | ボトルネック | Q見込み |
|------|-----------|------|-----------|------------|---------|
| beam-target | 固体ホウ素 | 束縛電子 | ~10⁻⁶ | 電子阻止能 100keV/μm | ~10⁻⁵ |
| beam-plasma | ホウ素プラズマ | 自由電子 | ~10⁻³ | ビーム-電子ドラッグ | ~10⁻² |
| **beam-beam（CBFR）** | ホウ素ビーム | 中和電子 | 高い | **ビーム熱化 + 電子ドラッグ** | **< 1（Rider）** |

CBFRは反応率こそ最も高い（共鳴エネルギーに最適化できるため）が、ビーム維持コストが核融合出力を食い尽くす。

### 全方式に共通する教訓

```
beam-target:  電子阻止能で止まる
beam-plasma:  電子ドラッグで止まる
beam-beam:    ビーム熱化 + 電子ドラッグで止まる
thermal:      P_ie + P_brem で止まる

全方式で電子がエネルギーを奪う。
方式を変えても Z_eff² = 18.8 は変わらない。
```

## 5. それでもCBFRから学べること

CBFRは失敗したが、以下の知見はTAEの現在のアプローチに生きている：

| CBFRの知見 | 現在のTAE方式への活用 |
|-----------|---------------------|
| FRCは大ラーモア半径イオンと相性が良い | NBI高速イオンのFRC閉じ込め |
| ビーム駆動でFRCを安定化できる | NBI駆動FRCの基盤 |
| 非マクスウェル分布は反応率を上げる | ビーム成分の核融合寄与 |
| 電子の問題は回避できない | αチャネリングへの注目 |

**CBFRは「正しい問題を見つけるための間違った答え」だった。**
