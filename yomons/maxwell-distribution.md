## マクスウェル分布とは

**定義**: 熱平衡にある粒子集団の速度分布。
「ある温度 T のとき、粒子がどの速度をどれくらいの確率で持っているか」を記述する。

$$
f(v) = 4\pi n \left(\frac{m}{2\pi k_B T}\right)^{3/2} v^2 \exp\left(-\frac{mv^2}{2k_BT}\right)
$$

- $n$: 粒子数密度
- $m$: 粒子の質量
- $T$: 温度
- $v$: 速さ

**ポイント**: 「温度」とは粒子の速度の散らばり具合を一つの数字で表したもの。
温度が高い → 速い粒子も遅い粒子もいて、平均速度が高い。

### 日常の類比

**満員電車の改札を出た人々の歩行速度。**

- ほとんどの人は「平均的な速さ」で歩く
- すごく遅い人（お年寄り）も少しいる
- すごく速い人（走ってるサラリーマン）も少しいる
- 十分な時間が経つと、自然にこのベル型分布に落ち着く

**放っておけば必ずこうなる**のがマクスウェル分布。熱力学第二法則の帰結。

### 核融合での意味

温度 T = 300 keV のイオンプラズマがマクスウェル分布のとき：

```
確率
 ↑
 │   ██
 │  ████
 │ ██████
 │████████░░            ← ガモフピーク（核融合が起こる速さ）
 │██████████░░░
 │████████████░░░░
 └──────────────────→ 速度（エネルギー）
   低い    平均    高い

 █ = 核融合しない粒子（大多数）
 ░ = 核融合できる粒子（ごく一部）
```

**ほとんどの粒子は核融合に寄与しない「無駄飯食い」。**
エネルギーを持っているのにクーロン障壁を越えられない速度で動いている。

## 非マクスウェル分布とは

**定義**: 熱平衡でない速度分布の総称。マクスウェル分布でないもの全て。

4サイクル案で狙うのは**ビーム的分布**：
イオンを特定のエネルギー（ガモフピーク付近の 500〜800 keV）に集中させた状態。

$$
f_{beam}(v) \approx n \cdot \delta(v - v_{Gamow})
$$

（理想的にはデルタ関数。実際にはある程度の幅を持つ）

### 日常の類比

**駅伝の選手たち。**

全員がほぼ同じペース（km 3分台）で走っている。
ふらふら歩いてる人はいないし、100m ダッシュしてる人もいない。
全員が「目的に最適な速度」に揃っている。

### 核融合での意味

電子ビームでイオンをガモフピーク付近に加速すると：

```
確率
 ↑
 │
 │
 │                 ██
 │                ████   ← ガモフピーク（全員ここに集中）
 │               ██████
 │              ████████
 └──────────────────→ 速度（エネルギー）
   低い    平均    高い

 █ = ほぼ全員が核融合できる速さ
```

**「無駄飯食い」がいない。同じ平均エネルギーでも核融合率が 2〜5 倍高い。**

$$
\langle\sigma v\rangle_{beam} > \langle\sigma v\rangle_{Maxwellian}
\quad \text{（同じ平均エネルギーでも）}
$$

## 対比表

| | マクスウェル分布 | 非マクスウェル分布（ビーム的） |
|--|--|--|
| **状態** | 熱平衡（放っておくとこうなる） | 非平衡（人為的に作る必要がある） |
| **速度の散らばり** | 広い（遅い〜速いまでばらばら） | 狭い（特定速度に集中） |
| **核融合に有効な粒子の割合** | 全体の数%以下 | ほぼ全員 |
| **P_fusion（同じ平均エネルギー比較）** | 基準 | 2〜5倍 |
| **日常の類比** | 改札を出た人々 | 駅伝の選手 |
| **維持コスト** | ゼロ（自然状態） | 高い（崩れるのを防ぐ必要あり） |
| **安定性** | 永久に安定 | 崩れる（→ 熱化） |
| **Rider（1995）の計算** | これを前提 | これが成立すれば Rider を回避 |

## 非マクスウェル分布の最大の問題: 熱化

非マクスウェル分布は**放っておくと崩れる**。

粒子同士がクーロン衝突するたびに速度がばらけ、最終的にマクスウェル分布に戻る。
駅伝の選手が走り終わった後、てんでばらばらに歩き出すようなもの。

### 熱化時間 τ_th

イオン-イオン間のクーロン衝突による熱化時間:

$$
\tau_{ii} \approx \frac{12\pi^{3/2} \varepsilon_0^2 m_i^{1/2} (k_B T_i)^{3/2}}{n_i Z^4 e^4 \ln\Lambda}
$$

- $\ln\Lambda \approx 10〜20$: クーロン対数
- $T_i$: イオン温度
- $n_i$: イオン密度
- $Z$: 電荷数

### p-B11 条件での概算

$T_i \sim 300$ keV、$n_i \sim 10^{25}$ m⁻³、$\ln\Lambda \sim 15$ として:

$$
\tau_{ii} \sim 10〜100 \text{ ns}
$$

### 4サイクルのタイムスケールとの比較

| プロセス | 時間スケール |
|---------|------------|
| 核融合反応そのもの | ~10⁻²² s |
| Zピンチ燃焼フェーズ | 1〜10 ns |
| **イオン-イオン熱化 τ_ii** | **10〜100 ns** |
| 電子-イオン熱平衡化 τ_eq | ~1 μs |

**燃焼（1〜10 ns）< 熱化（10〜100 ns）**

→ 燃焼時間が熱化時間より短ければ、非マクスウェル分布は燃焼中に維持される。
→ **ギリギリ成立する可能性がある。ただしマージンは 1〜10 倍程度しかない。**

### タイムスケールの図

```
          1 ns     10 ns    100 ns    1 μs
──────────┼────────┼────────┼─────────┼──→
          │        │        │         │
          │◄──燃焼──►│       │         │
          │        │◄──τ_ii（熱化）──►│
          │        │        │         │◄── τ_eq
          │        │        │
          窓がある？  重なってる？
```

**窓は存在するが狭い。** 燃焼が 1 ns で終わり、熱化が 100 ns なら余裕がある。
燃焼が 10 ns かかり、熱化が 10 ns なら間に合わない。

## 要問

🔴 **非マクスウェル分布を燃焼完了まで維持できるか？**

これが抜け道①の成否を決める核心の問い。

**具体的に確認すべきこと:**

1. **τ_ii の正確な値**: 密度・温度・組成（p + B11）でのイオン-イオン熱化時間
2. **燃焼時間の正確な値**: 与えられた密度・温度での p-B11 反応の燃焼時間
3. **窓の幅**: τ_burn < τ_ii がどの程度のマージンで成立するか
4. **電子ビームで本当にビーム的分布を作れるか**: ビーム入射→プラズマ中での分布形成過程

**もし τ_burn ≈ τ_ii（マージンなし）なら：**
- 非マクスウェル分布の効果は部分的にしか得られない
- P_fusion の改善は 2〜5 倍ではなく 1.2〜2 倍程度に縮小
- 3つの抜け道の合計が P_fusion/P_brem ≈ 2.3 を下回る可能性

**もし τ_burn << τ_ii（十分なマージン）なら：**
- 非マクスウェル分布の効果がフルに効く
- P_fusion/P_brem ≈ 2.3 以上が期待できる

## Rider（1995）が正しい場合と間違っている場合

| Rider の前提 | 成立する場合 | 崩れる場合 |
|-------------|------------|-----------|
| マクスウェル分布 | Ti/Te ≈ 300 必要。p-B11 は絶望的 | 非マクスウェルで P_fusion 2〜5 倍。希望あり |
| 定常状態 | 制動放射が永遠に蓄積 | パルスで逃げ切れる。4サイクルの根拠 |

**4サイクル案は Rider の両方の前提を崩すことに賭けている。**
そしてその賭けの中で最も重要なのが「非マクスウェル分布の維持」。
