## これは何？

素人エンジニアが Claude と一緒に「p-¹¹B 核融合で発電できないか」を真剣に考えた記録。

戦争ってエネルギー不足が根本にある気がして。  
そのへんにある水素とホウ素を核融合させるだけで無限にエネルギーが出るなら、世界変わるよね。  
バリムズい（SSS難易度）らしいけど、構造を理解するだけでも価値がある。

## p-B11 とは

```
p  +  ¹¹B  →  3 × ⁴He  +  8.7 MeV
陽子   ホウ素11    ヘリウム×3
```

| 比較項目 | D-T（主流） | p-¹¹B（究極） |
|---------|-----------|------------|
| 中性子 | エネルギーの 80% が中性子 | ほぼゼロ |
| 放射性廃棄物 | 多い | ほぼなし |
| 燃料 | 三重水素は希少・放射性 | 水素 + 天然ホウ素（安価・豊富）|
| 直接発電 | 困難 | α粒子で原理的に可能 |
| 難易度 | 高い | 極めて高い |

```
🟢 解決済み    2/13  タイミング同期、燃料供給
```

「電子を冷たく保ちながらイオンだけを数十億℃に維持できるか」が人類未解決の最重要課題。

### yomons/（要問）

GitHub の Issues 機能と被るため「要問（ようもん）」をローマ字で `yomons` にした。  
要問の定義・フレームワークは [`yomons/README.md`](yomons/README.md) を参照。

| ファイル | 内容 |
|---------|------|
| `overview.md` | 要問ツリー全体・解決状況・クリティカルパス |
| `L1-physics.md` | クーロン障壁・制動放射・ローソン基準 |
| `L2-engineering.md` | タイミング同期・繰り返し耐久・統合設計 |
| `L3-materials.md` | 壁劣化・プラズマ汚染・液体金属壁 |
| `L4-systems.md` | 直接発電・エネルギー収支・燃料供給 |

### approaches/（アプローチ）
