# 4サイクル核融合エンジン — コンセプト概要

## 発想の起点

車のエンジンは「吸気→圧縮→燃焼→排気」の4ストロークで複数の物理的制約を**時系列で分離**している。  
核融合の問題も「全部同時に満たそうとするから難しい」のではないか？  
フェーズを分ければ、各フェーズで個別の最適化ができる。

---

## 4フェーズの設計

### Phase 1: 吸気（電子ビーム集束）

```
目的: 陽子とホウ素イオンを炉の中心に集める
手段: 電子ビームで中心部に負の電位井戸を形成
     → 陽子（p⁺）とホウ素イオン（B⁵⁺）が電気的に引き寄せられる
時間: 100〜500 ns（再結合が起きる前に終える）
```

**ポイント**: 電子とイオンが「仲良くなる（中性化）」前に次フェーズへ移行するのがキモ。  
再結合の時定数（μs オーダー）より圧縮時間（ns オーダー）の方が速いので原理的に逃げ切れる。

**副次効果**: 電子ビームはイオンを加熱せずに集めるだけなので、
電子温度を低く保ちながらイオン密度だけを上げられる → 制動放射抑制に有利。

**非マクスウェル分布の形成**: 電子ビームでイオンをガモフピーク付近のエネルギーに集中させると、
熱平衡（マクスウェル分布）ではなくビーム的分布が生まれる。
同じ平均エネルギーでも核融合率が 2〜5 倍高くなり、Rider（1995）の限界を緩和できる可能性がある。

**ラーモア半径差による電子排除**: 磁場中で電子（r_L ~0.1mm）とイオン（r_L ~数mm〜cm）のラーモア半径が
2桁異なることを利用し、電子だけを選択的に排除・冷却できる（ホールスラスタ等で実績あり）。

> 詳細: [`../../yomons/larmor-radius.md`](../../yomons/larmor-radius.md) / [`../../yomons/ti-te-escape.md`](../../yomons/ti-te-escape.md)

### Phase 2: 圧縮（Zピンチ爆縮）

```
目的: プラズマを超高温・超高密度に圧縮
手段: 外側電極にMA級電流をナノ秒パルスで流す
     ローレンツ力（J×B）がプラズマを内側に圧縮
時間: 10〜50 ns
```

**物理**: 断熱圧縮により $T \propto \rho^{\gamma-1}$ の関係で温度が急上昇。
Zピンチ電流はイオンを直接圧縮→加熱するため、電子温度より先にイオン温度が上がる。
これが二温度プラズマ（Ti >> Te）の自然な形成につながる。

**超短パルスによる Te 抑制**: オーム加熱（$P_{ohm} = J^2 \eta \Delta t$）は時間に比例するため、
パルスを数 ns に短くすると電子加熱を最小化できる。断熱圧縮は圧縮比で決まり時間によらないため、
超短パルスでは「イオンだけ加熱、電子は冷たいまま」が成立しやすい。

**課題**: レイリー・テイラー不安定性。完全球対称でない場合、界面が崩れる。
→ Phase 1 の電子ビームで事前に中心密度を高めておくことで緩和できる可能性。

### Phase 2b: 電子排出（線の段階で電子を捨てる）

```
目的: 制動放射の原因となる電子を選択的に除去
手段: 円筒圧縮後、端を数ns開けて電子を軸方向に流出させる
     正電位電極で電子を引き寄せ、イオンを押し戻す
時間: 2〜5 ns
```

**物理的根拠**: 線状プラズマでは磁場が軸方向に整列。
電子はラーモア半径が小さく磁力線に束縛されるため、軸方向にしか動けない。
電子の軸方向走行時間（~1 ns）はイオン（~100 ns）の 1/100。
→ 端を数 ns 開けるだけで**電子だけが出てイオンは残る**。

**現実的な効果**: アンビポーラ電位の壁により n_e の除去は 10〜20% が限界。
P_brem を 0.64 倍にする程度で、単体では制動放射問題を解決しない。
抜け道①②の補助（ブースター）としての位置づけ。

**致命的な矛盾**: 電子はZピンチ電流の担い手。電子を抜くと Phase 2c の圧縮ができなくなる可能性がある。

**それでもルートC（体→線→点）でしかできない操作。**
球対称爆縮（ルートD）では磁場の方向性がなく電子排出自体ができない。

### Phase 2c: 軸方向圧縮（線→点）

```
目的: 電子が少ない状態で最終圧縮
手段: エンドプラグまたは軸方向磁場勾配で軸方向に圧縮
時間: 5〜20 ns
```

電子排出後のプラズマは正味正電荷を持つ。この状態でのZピンチ安定性は未検証。

### Phase 3: 燃焼（核融合バースト）

```
目的: p-¹¹B 核融合反応を起こす
反応: p + ¹¹B → 3 × ⁴He + 8.7 MeV
時間: 1〜10 ns（反応タイムスケール）
```

**なぜ p-B11 か**:
- 中性子がほぼゼロ → 炉壁の放射化なし
- 生成物がα粒子（⁴He²⁺）のみ → 直接電気変換が可能
- 燃料が水素＋天然ホウ素 → 安価・豊富・非放射性

**必要条件**: イオン温度 ~50〜300億℃（500 keV〜3 MeV）
→ これがD-Tの約30〜100倍高く、最大の物理的障壁。

**タイムスケールの根拠**: 燃焼フェーズ（1〜10 ns）は電子-イオン熱平衡化時間（τ_eq ~1 μs）の
1000 分の 1 しかない。つまり「燃やし切ってから電子が温まる」＝制動放射が本格化する前に核融合を終える。
これが4サイクル方式の物理的根拠。

> 詳細: [`../../yomons/fusion-timescales.md`](../../yomons/fusion-timescales.md)

### Phase 4: 排気（α粒子回収＋リセット）

```
目的: 核融合生成物の回収と炉のリセット
手段:
  ① α粒子（⁴He²⁺）を外側のコレクターリングで受け止める
  ② 磁場または静電デセレレータで運動エネルギーを直接電気変換
  ③ 残留ガス（He）を磁場を弱めて自然膨張・排出
  ④ 液体金属壁をパルス間に循環・リフレッシュ
時間: 100〜1000 ns
```

**直接発電の意義**: 熱→蒸気→タービン→発電機という通常の経路を省略。  
理論変換効率 60〜90%（熱機関のカルノー限界 ~40% を大幅に超える）。

## 既存研究との関係

この発想に最も近い既存アプローチ：

- **MagLIF（Sandia国立研究所）**: 磁化ライナー慣性核融合。Zピンチ爆縮＋レーザー予備加熱。D-Tで原理実証済み。
- **Helion Energy**: FRCプラズマを磁気圧縮。直接発電を目指す。Microsoftと電力購入契約。
- **TAE Technologies**: FRC＋段階加熱でp-B11を目標。最も資金力があるp-B11挑戦者。

この4サイクル案は「MagLIFに電子ビーム集束フェーズ + 電子排出フェーズを追加した形」に近い。
電子ビームによる前段集束と、線段階での電子排出が二温度プラズマ形成に有利という点は従来にない視点。

## 未解決の核心問題

1. **制動放射収支**: Rider（1995）によればマクスウェル分布前提で **Ti/Te ≈ 300** が必要。抜け道①②＋③補助の組み合わせで P_fusion/P_brem ≈ 2.3（ギリギリ NET）。③の電子排出はアンビポーラ電位の壁とZピンチ矛盾により効果は限定的
2. **繰り返し耐久性**: 高繰り返しパルスに耐える電極・壁材料が存在しない
3. **スケールアップ**: 1パルスのエネルギーを増やさないと補機電力を賄えない
4. **最重要未解決実験**: 電子ビーム＋超短Zピンチで非マクスウェル分布のイオンを作れるか — これが確認されれば議論が大きく動く
