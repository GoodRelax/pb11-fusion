# 4サイクル核融合エンジン — コンセプト概要

## 発想の起点

車のエンジンは「吸気→圧縮→燃焼→排気」の4ストロークで複数の物理的制約を**時系列で分離**している。  
核融合の問題も「全部同時に満たそうとするから難しい」のではないか？  
フェーズを分ければ、各フェーズで個別の最適化ができる。

---

## 4フェーズの設計

### Phase 1: 吸気（電子ビーム集束）

```
目的: 陽子とホウ素イオンを炉の中心に集める
手段: 電子ビームで中心部に負の電位井戸を形成
     → 陽子（p⁺）とホウ素イオン（B⁵⁺）が電気的に引き寄せられる
時間: 100〜500 ns（再結合が起きる前に終える）
```

**ポイント**: 電子とイオンが「仲良くなる（中性化）」前に次フェーズへ移行するのがキモ。  
再結合の時定数（μs オーダー）より圧縮時間（ns オーダー）の方が速いので原理的に逃げ切れる。

**副次効果**: 電子ビームはイオンを加熱せずに集めるだけなので、  
電子温度を低く保ちながらイオン密度だけを上げられる → 制動放射抑制に有利。

---

### Phase 2: 圧縮（Zピンチ爆縮）

```
目的: プラズマを超高温・超高密度に圧縮
手段: 外側電極にMA級電流をナノ秒パルスで流す
     ローレンツ力（J×B）がプラズマを内側に圧縮
時間: 10〜50 ns
```

**物理**: 断熱圧縮により $T \propto \rho^{\gamma-1}$ の関係で温度が急上昇。  
Zピンチ電流はイオンを直接圧縮→加熱するため、電子温度より先にイオン温度が上がる。  
これが二温度プラズマ（Ti >> Te）の自然な形成につながる。

**課題**: レイリー・テイラー不安定性。完全球対称でない場合、界面が崩れる。  
→ Phase 1 の電子ビームで事前に中心密度を高めておくことで緩和できる可能性。

---

### Phase 3: 燃焼（核融合バースト）

```
目的: p-¹¹B 核融合反応を起こす
反応: p + ¹¹B → 3 × ⁴He + 8.7 MeV
時間: 1〜10 ns（反応タイムスケール）
```

**なぜ p-B11 か**:
- 中性子がほぼゼロ → 炉壁の放射化なし
- 生成物がα粒子（⁴He²⁺）のみ → 直接電気変換が可能
- 燃料が水素＋天然ホウ素 → 安価・豊富・非放射性

**必要条件**: イオン温度 ~50〜300億℃（500 keV〜3 MeV）  
→ これがD-Tの約30〜100倍高く、最大の物理的障壁。

---

### Phase 4: 排気（α粒子回収＋リセット）

```
目的: 核融合生成物の回収と炉のリセット
手段:
  ① α粒子（⁴He²⁺）を外側のコレクターリングで受け止める
  ② 磁場または静電デセレレータで運動エネルギーを直接電気変換
  ③ 残留ガス（He）を磁場を弱めて自然膨張・排出
  ④ 液体金属壁をパルス間に循環・リフレッシュ
時間: 100〜1000 ns
```

**直接発電の意義**: 熱→蒸気→タービン→発電機という通常の経路を省略。  
理論変換効率 60〜90%（熱機関のカルノー限界 ~40% を大幅に超える）。

## D-T との根本的な違い

| 項目 | D-T 主流方式 | p-B11 4サイクル |
|------|------------|----------------|
| 閉じ込め | 連続的磁気閉じ込め | パルス式（慣性+磁気ハイブリッド） |
| 温度 | ~1億℃ | ~数十億℃ |
| 中性子 | エネルギーの80% | ほぼゼロ |
| 発電 | 熱サイクル | 直接変換 |
| 材料劣化 | 中性子損傷が主 | α粒子スパッタ・熱衝撃が主 |
| 燃料 | 三重水素（希少・放射性） | 水素＋天然ホウ素 |

## 既存研究との関係

この発想に最も近い既存アプローチ：

- **MagLIF（Sandia国立研究所）**: 磁化ライナー慣性核融合。Zピンチ爆縮＋レーザー予備加熱。D-Tで原理実証済み。
- **Helion Energy**: FRCプラズマを磁気圧縮。直接発電を目指す。Microsoftと電力購入契約。
- **TAE Technologies**: FRC＋段階加熱でp-B11を目標。最も資金力があるp-B11挑戦者。

この4サイクル案は「MagLIFに電子ビーム集束フェーズを追加した形」に近い。  
電子ビームによる前段集束が二温度プラズマ形成に有利という点は従来にない視点。

## 未解決の核心問題

1. **制動放射収支**: Ti/Te = 10〜20 を維持できれば理論上NETになる計算は存在するが未実証
2. **繰り返し耐久性**: 高繰り返しパルスに耐える電極・壁材料が存在しない
3. **スケールアップ**: 1パルスのエネルギーを増やさないと補機電力を賄えない
